一张 P0 游戏 UI 组件 sprite sheet，2 行 x 2 列，共 4 个独立 UI 组件。用于《元生的六年级》横版格斗 RPG，不是宣传图。

== 风格 ==
像素风 UI，32-bit JRPG / GBA-to-DS 质感，2008 年中国农村小学作业本、成绩单、纸条、布袋的怀旧文具风。清晰像素边缘，1px 深色描边。背景纯洋红 #FF00FF 用于透明抠图。

== 切片用途 ==
这张图会被按 2 行 x 2 列均分裁切成 4 个独立 UI 组件。每个组件必须完整收在自己格子的中央区域，不能跨格。如果画绿色等分线，绿色线只能穿过纯洋红背景，不能切到 UI。

== 布局 ==
- 2 行 x 2 列，整图宽高比约 1:1。
- 背景 100% 纯洋红 #FF00FF。
- 不要分隔线、不要边框、不要编号、不要可读文字。

== 四个 UI 组件 ==
第 1 行第 1 列：对话框 UI。浅黄作业本格子纸质感，蓝色横线，左侧 48x48 头像预留框，右侧大文字区域，铅笔线边框。不要真实文字。
第 1 行第 2 列：选择分支面板。2-4 张纸条纵向叠放，高亮纸条微微翘起，纸张边缘铅笔线。
第 2 行第 1 列：属性面板。学生证/成绩单样式，四项属性用铅笔涂格子表示，不要写清晰汉字。
第 2 行第 2 列：钱包 UI。小旧布袋图标，旁边有数字区域占位但不要生成数字，表现空袋和鼓袋两个小状态。

== 禁止项 ==
- 不要可读文字、字母、数字、品牌 logo、水印、签名。
- 不要现代扁平 UI、霓虹科幻、玻璃拟态。
- 不要白底；所有组件外空白必须纯洋红 #FF00FF。
