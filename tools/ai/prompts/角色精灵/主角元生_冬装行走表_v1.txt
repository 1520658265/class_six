单张 RPG 角色 walking sprite sheet：主角元生冬装。用于 Godot 中按整张 raw sheet 配置行走动画，不要求切成单帧。

== 风格 ==
32-bit JRPG / PS1-GBA-to-DS 像素风，清晰 chunky pixel，1px 深色描边，2008 年安徽大别山区农村小学。背景纯洋红 #FF00FF。

== 布局 ==
- 4 行 x 4 列 sprite sheet。
- 行：朝下 / 朝左 / 朝右 / 朝上。
- 列：idle / walk1 / walk2 / walk3。
- 每格是独立 32x32 逻辑角色帧，角色完整收在格内。
- 背景和格间空白统一纯洋红 #FF00FF，不要黑格线、编号、标签。

== 角色 ==
元生，11 岁，偏瘦偏矮，皮肤偏黑，母亲剪的板寸/毛寸不齐整。冬装：深蓝或墨绿色厚棉袄，袖口磨出棉花，里面露一点薄毛衣，旧布棉鞋，手指有冻疮红肿。背单肩斜挎帆布包，磨得起毛。

== 动作 ==
16 帧里发型、肤色、棉袄、书包必须一致。步行帧只变化腿和手臂小幅摆动，身体最多上下 1 像素。

== 禁止项 ==
不要现代羽绒服品牌、拉杆箱、智能手表、文字、标签、编号、水印、白底、灰底。
