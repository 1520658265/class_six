单个 RPG 角色概念预览图：曹正东奶奶。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 58%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 大别山区农村老人气质，朴素、佝偻、生活艰难但不猎奇。

== 角色形象 ==
- 曹正东奶奶，70 多岁，极瘦佝偻，脸上皱纹深，面色蜡黄。
- 头包黑色方巾，灰白头发少量露出。
- 神情疲惫、沉默、咳嗽感，但不要恐怖化。

== 服装 ==
- 深色对襟棉袄 #3a3834 或 #2f342f，旧而厚。
- 深色宽裤，旧布鞋。
- 可以拄一根简单木拐杖。

== 配色 ==
- 皮肤 #b88967
- 头巾黑灰 #1f1f1e
- 主描边 #2a2520
- 对襟棉袄深灰褐 #3a3834
- 对襟棉袄深绿黑 #2f342f

- 木拐杖 #6a4b32

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母。
- 不要画成巫婆、乞丐、神怪或恐怖角色。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
