一张 P1 UI 组件 sprite sheet，1 行 x 2 列：心结提示 UI、钱包羞耻度 UI。

== 风格 ==
像素风 UI，作业本/日记本手写墨迹风，2008 年中国农村小学怀旧调，背景纯洋红 #FF00FF。

== 切片用途 ==
按 1 行 x 2 列均分裁切。每个 UI 完整居中，绿色等分线只能穿过洋红背景。

== 两个组件 ==
第 1 列：心结提示 UI。屏幕下方浮现一行手写体小字的视觉，用墨色模糊笔画和渐淡效果表示，不要可读文字，旁边有小小裂心图标。
第 2 列：钱包羞耻度 UI。暗红手写提示视觉，口袋空空小图标闪烁感，墨色渐淡，不要可读文字。

== 禁止项 ==
不要可读文字、数字、现代 UI、霓虹、白底、灰底、水印。
