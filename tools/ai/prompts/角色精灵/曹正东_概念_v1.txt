单个 RPG 角色概念预览图：曹正东，丁埠小学六年级男生，贯穿剧情的冲突型角色。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 68%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后中国农村小学怀旧调，压迫感强、但仍是小学生。

== 角色形象 ==
- 曹正东，11 岁左右，但看起来比实际年纪更大一圈，偏壮偏高，肩膀宽，脸略方。
- 皮肤黝黑，脸上有一道旧疤痕，眉眼压迫感强。
- 头发黑短，乱但不长，站姿带一点吊儿郎当。
- 神情凶、硬、倔，像经常拦路的人，但仍要保留小学生比例。

== 服装 ==
- 旧深色棉袄或脏旧夹克，颜色偏墨绿、深褐、脏灰，不合身。
- 深色长裤，裤脚沾灰，鞋子旧。
- 整体要显得脏旧，但不是成年混混。

== 道具 ==
- 不要固定武器；可以空手、手插裤兜、或者微微伸手拦人。
- 不要刀棍枪械，不要夸张黑帮感。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要潮牌图案。
- 不要画成成年人，不要画成黑帮老大。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
