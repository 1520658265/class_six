一张横版格斗 RPG 场景背景：丁埠小学小卖部内景/柜台窗口，2008 年中国农村小学小卖部。用于游戏场景背景，不是 UI，不是插画海报。

== 风格 ==
32-bit JRPG / PS1-GBA-to-DS 像素风，横版 2D 场景，清晰 chunky pixel，1px 深色描边，怀旧褪色调。不要照片感，不要 3D，不要平滑厚涂。

== 构图 ==
- 宽幅横版场景，宽高比 16:9。
- 横向柜台视角，柜台在中前景，后方是货架。
- 无人物，柜台前留出可放角色的地面区域。

== 场景内容 ==
- 宿舍楼一楼改的小卖部，一扇窗口当柜台。
- 水泥台面柜台，台面上摆翻烂的赊账本和圆珠笔。
- 一排大玻璃罐，里面有散称零食、糖、唐僧肉等，用色块表现不要文字。
- 墙上木架子摆方便面、文具、洗衣粉、杂货。
- 小型滚轴烤肠机，4-6 根红色淀粉肠。
- 角落有冰柜，光线偏暗，柜台窗口透光加一盏白炽灯泡。

== 禁止项 ==
- 不要人物，不要现代便利店，不要扫码牌，不要 LED 招牌。
- 不要清晰可读文字、英文字母、品牌 logo、水印、签名。
