# grass_cliff 切片实验

从 OpenGameArt seamless-64px-rpg-tiles 中抽 grass_cliff 切片做的 tileset 实验。

包含：
- `slice_grass_cliff.py`、`extract_clusters.py`、`build_*.py` — 切片与 atlas 构建脚本
- `atoms_grass_cliff/` — 单 tile atom 输出
- `cluster_*.png`、`pick_mosaic.png`、`grid_labeled_*.png` — 调试可视化
- `village_grass_cliff_tileset.tres` — Godot tileset 资源（实验阶段）
- `village_grass_cliff_tileset_preview.png`、`_probe.json` — 预览与探针

非正式美术资产。
