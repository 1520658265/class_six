一张 1 行 x 3 列的横版 RPG 冬季变体场景组：教室煤炉、宿舍加被、山路薄雪。

== 风格 ==
32-bit JRPG / PS1-GBA-to-DS 像素风，横版 2D 场景，清晰 chunky pixel，1px 深色描边，冬季灰冷怀旧调。

== 切片用途 ==
按 1 行 x 3 列均分成 3 个独立场景。每个场景完整落在自己格子内。背景空白纯洋红 #FF00FF，虚拟网格不画出来。

== 三个格子 ==
第 1 列：二班教室冬季变体。教室中间铸铁煤炉，铁皮烟囱从窗户伸出，窗霜，煤渣，旧课桌椅。
第 2 列：男生宿舍冬季变体。铁架床、厚被子、蚊帐收拢、窗缝冷光，床边热水瓶和脸盆。
第 3 列：山路薄雪。黄土碎石山路覆薄雪，枯树、松树、茶园灰绿，远山冷雾。

== 禁止项 ==
不要人物、清晰文字、现代暖气、空调、白底、灰底、边框、水印。
