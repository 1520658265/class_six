单个 RPG 角色概念预览图：元生的母亲。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 64%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后大别山区农村母亲气质，朴素、能干、温和。

== 角色形象 ==
- 元生的母亲，38-40 岁，中国山区农村妇女，中等身材，劳作感明显。
- 黑发用黑色发卡别在耳后或低低扎起，发型朴素。
- 脸上有疲惫但温和的神情，眼神关心孩子，不要过度悲情。
- 手部可略粗糙，有裂口和茧的感觉。

== 服装 ==
- 碎花棉布上衣，底色旧紫红 #8b4a5a 或深蓝灰 #4f5965，小花点缀。
- 系一条深色围裙 #4a463f。
- 深色长裤，朴素布鞋。

== 道具 ==
- 可以手里拿一个搪瓷碗、布包或几张折好的纸币，表现“塞零钱/端碗”的生活感。
- 不要现代手提包，不要手机，不要时尚首饰。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要精致妆容。
- 不要画成城市太太、服务员或古装人物。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
