单个 RPG 角色概念预览图：张磊，一班男生，留守儿童但成绩好。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 63%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后中国农村小学怀旧调，朴素、安静、有距离感。

== 角色形象 ==
- 张磊，11 岁左右，一班男生，外观和元生一样朴素，但更沉默内向。
- 黑色短发，脸偏瘦，眼神常看向远处，像在等外出打工父母消息。
- 小学生比例，约 3.7 到 3.9 头身。

== 服装 ==
- 洗旧的灰蓝长袖上衣 #6b7f90，袖口略旧。
- 深色长裤，旧布鞋或解放鞋。
- 戴红领巾 #c83828，但系得不太精神。

== 道具 ==
- 可以手里拿旧书或小布袋，不要明显贵重物品。
- 不要手机、MP3、现代电子设备。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母。
- 不要画得过度悲惨或邋遢；要可信、克制。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
