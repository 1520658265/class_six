单张 RPG Maker 风格角色 walking sprite sheet 图，主角“元生”（1996 年生，2008 年小学六年级男生），用于 Godot 顶视 RPG 地图。

== 画布布局 ==
画布是一张正方形图，划分成严格对齐的 4 列 × 4 行 = 16 个完全相同大小的正方形格子，格子之间以及画布外圈都是 8 像素宽的纯洋红 #FF00FF gutter（用于后处理 chroma-key 抠图）。每个格子内部都是一个独立的 32×32 像素帧，chunky pixel 风格放大渲染。
不要画格线、不要画格子编号、不要画任何文字标签、不要画外边框、不要画水印、不要画辅助说明。

== 行列含义 ==
- 第 1 行 = 朝下（面对镜头）
- 第 2 行 = 朝左
- 第 3 行 = 朝右
- 第 4 行 = 朝上（背对镜头）
- 第 1 列 = idle 站立
- 第 2 列 = 步行循环 1（左脚向前）
- 第 3 列 = 步行循环 2（双脚并立中间帧）
- 第 4 列 = 步行循环 3（右脚向前）

== 方向正确性要求 ==
- 第 1 行 4 帧必须全部朝下，只允许脚和左手变化。
- 第 2 行 4 帧必须全部朝左，必须是清楚的侧身 profile。
- 第 3 行 4 帧必须全部朝右，必须是清楚的侧身 profile。
- 第 4 行 4 帧必须全部朝上，只能看到后脑勺和背部，不能看到正脸。
- 同一行内发型、衣服、配件、比例必须完全一致；只能腿和左手摆动。

== 角色形象 ==
元生，12 岁左右，2008 年中国农村小学六年级男生，约 3.5 到 3.8 头身；头明显偏大，身体短，比例紧凑。
他不是自卑敏感型，而是爱装酷、要面子、嘴上有点拽的小学生。
视觉上要比普通同学更有主角感，但仍然是校园少年，不是成人潮人。

- 头发：黑色 #1a1410，校园少年版杀马特斜刘海，从右上斜向左下盖住右眼，露出左眼；刘海边缘分层，发梢略微外翘。
- 脸：脸型略窄但保留稚气，肤色 #e8b890，左眼半眯，眉毛略斜，有点不服气、有点装帅；不要苦相，不要低头躲闪。
- 上衣：米白色短袖 T 恤 #f2ead8，深蓝色领口和袖口 #1f2c46，胸前一个很小的深蓝抽象闪电/斜杠图案，无文字。
- 红领巾：鲜红 #c83828，结打得小一点；朝上方向只用 1-2 个红色像素表现绕后颈的两端。
- 下装：浅蓝短牛仔裤 #5f8fb8，裤脚卷边，深蓝描边 #2a3850。
- 鞋子：白色低帮运动鞋 #f0f0e8，黑色鞋底 #2a2520。
- 不背任何包：不要斜挎包、不要双肩包、不要书包带。
- 右手：所有 16 帧右手都插在短牛仔裤口袋里，不摆动。
- 左手：露在外面，手腕有黄色塑料手环 #f0c828；idle 自然垂下，步行帧随动作摆动。

== 步行帧细节 ==
- 步行循环只摆动腿和左手，右手插兜不动，头部和身体保持稳定。
- 第 2 列：左脚迈出。
- 第 3 列：双脚并立中间帧。
- 第 4 列：右脚迈出。
- 朝左/朝右方向的步行必须看到腿前后交替。
- 朝上方向能看到后脑勺、短袖背面、牛仔短裤背面、后颈红领巾像素。

== 风格规范 ==
- 16-bit JRPG 顶视半正面像素画风，适合后续切成 32x32。
- 不允许 anti-aliasing、不允许平滑渐变；只允许块状色块和少量 1 像素 dithering 阴影。
- 1px 深炭灰 #2a2520 描边只用在角色轮廓和服装分块上。
- 光源统一左上 45°。
- 角色脚底贴近 cell 下沿。

== 全图严格要求 ==
- 每一帧都必须严格停留在自己的 32×32 格子内，不能跨格。
- 16 个格子每个都必须有完整角色。
- 不要在画布上添加任何其它角色、动物、文字、UI、签名、水印、装饰边框。
- 不允许出现斜挎包、双肩包、书包带、手机、耳机、电子屏幕、现代品牌 logo、英文字母。
- 不要把元生画成苦相、胆怯、驼背、缩肩；他应该是一个爱装酷的小学生。
