单个 RPG 角色概念预览图：王炎，丁埠小学六年级男同学。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 65%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后中国农村小学怀旧调，带一点机灵和少年感。

== 角色形象 ==
- 王炎，11 岁左右，中国农村小学六年级男生，瘦长脸，眼睛灵活，神情机灵。
- 头发黑短，略乱但不邋遢。
- 早期气质像同宿舍同学，后期会更“城里货”一点，但这张先做基础版，保留一点自信和嘴硬。

== 服装 ==
- 上衣偏浅色 T 恤或薄外套，颜色可用灰白、浅蓝、暗红小拼色，整体朴素但略比主角新一点。
- 深色牛仔裤或深蓝长裤，鞋子可偏白。
- 不要夸张潮牌，不要成人化。

== 道具 ==
- 手里拿悠悠球或让悠悠球自然垂在手边，颜色塑料感强。
- 不要电子屏、手机、耳机、太现代的配饰。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要潮牌图案。
- 不要把角色画成坏学生混混，也不要画得过分乖巧。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
