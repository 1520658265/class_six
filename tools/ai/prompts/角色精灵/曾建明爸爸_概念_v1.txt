单个 RPG 角色概念预览图：曾建明爸爸。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 66%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年古碑镇教师/干部家庭气质，干净、稳定、条件较好但不奢华。

== 角色形象 ==
- 曾建明爸爸，40 岁左右，教师气质，戴眼镜，身形中等。
- 黑色短发，神情温和、讲道理。
- 皮肤较白净，站姿端正。

== 服装 ==
- 白衬衫或浅蓝衬衫 #e8eef2，深色西裤 #30343a。
- 可穿灰色毛背心或薄夹克，整体整洁。
- 黑色皮鞋或干净布鞋。

== 道具 ==
- 手里拿书本、报纸或教案，暗示文化家庭。
- 不要手机、笔记本电脑、现代商务包。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母。
- 不要画成老板、官员、商务精英或校长。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
