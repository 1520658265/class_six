一张 RPG 对话系统用的角色半身像 sprite sheet：鲍先进，同一个角色的 5 个克制写实表情，另有 1 个空白格。不是全身行走表，不是 UI 面板，不是展示海报。

== 风格声明 ==
32-bit JRPG era pixel art，PS1 / GBA-to-DS RPG 质感，清晰 chunky pixel，1px 深炭灰或深棕描边，有限但丰富的怀旧色板，轻微抖动阴影。不要平滑厚涂，不要 3D，不要照片感，不要 anti-aliasing。
时代气质是 2008 年前后中国农村小学，温暖、朴素、略晒旧。

== 用途声明 ==
这张图会被按 2 行 x 3 列均匀切割，每个格子被切出来后单独作为一张对话肖像贴图使用。每个半身像必须完整落在自己那一格的方形区域内，头发、肩膀、课本、烟雾、描边都不能越过自己的格子。角色外缘距离虚拟格子边界至少 25% 安全距离，相邻格子之间必须能看到连续的纯洋红空白。

== 画布布局 ==
- 2 行 x 3 列虚拟网格，共 6 个方形格。
- 每个格子逻辑规格 64x64，正方形 1:1。
- 整张图宽高比约 3:2。
- 背景是 100% 连续纯洋红 #FF00FF，虚拟网格不画出来。
- 不要分隔线、不要分隔条、不要 tile 边框、不要整张图外框。

== 角色外观锁定 ==
参考身份：assets/art/characters/baoxianjin/raw/char_baoxianjin_concept_v2.jpg
鲍先进，45-50 岁，中国农村小学六年级班主任/数学老师。半身像只画头、肩、少量胸口。
- 方脸，宽颧骨，皮肤黝黑粗糙 #7f5a42。
- 黑短硬寸头 #1a1410，不戴眼镜，不留长胡子。
- 粗眉、额头纹、法令纹明显，嘴角自然下垂。
- 深蓝黑外套 #34373d 或墨绿外套 #3e4a3a，领口有一点粉笔灰 #d8d0b8。
- 主描边 #2a2520。
- 同一角色 5 个表情里脸型、发型、皱纹、外套领口必须一致，只改变表情和小动作。

== 6 个格子，从左到右、从上到下 ==
第 1 行第 1 格，严肃：眉毛压住，眼睛直视略向下，嘴角自然下垂，像在讲台上盯人。
第 1 行第 2 格，凶：眉头收紧，眼睛更窄，法令纹加深，但不要变成恶人或怪物。
第 1 行第 3 格，笑：嘴角只轻微上扬，眼角皱纹放松，仍保持中年老师的压场感，不露夸张大牙。
第 2 行第 1 格，红眼：眼白略带暗红 #9b4d4d，眼眶疲惫，嘴唇绷住，像熬夜批作业后强撑。
第 2 行第 2 格，抽烟：一手在胸前夹细烟，灰白烟雾 #c8c0b0 只画 2-3 小段，烟雾完全收在本格中央，不能越界。
第 2 行第 3 格，空白：这一格必须完全空白。只有纯洋红 #FF00FF 填充，没有任何图形、点、线、十字、占位符。绝对什么都不画。

== 强禁项 ==
- 不允许文字、字母、数字、姓名标签、图例、编号、水印、签名。
- 不允许 UI 框、对话框、边框、tile 描边框、画布外框。
- 不允许大汗珠、椭圆 ○○ 惊讶眼、波浪嘴、夸张漫画符号、魔法符号。
- 不允许烟雾、手、肩膀、描边超出自己的格子。
- 不允许白底、灰底、纸张底；所有空白必须是纯洋红 #FF00FF。
- 不允许 anti-aliasing、模糊、柔焦、照片质感、3D 渲染。
