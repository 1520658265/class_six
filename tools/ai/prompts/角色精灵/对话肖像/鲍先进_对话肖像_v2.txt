一张 RPG 对话系统用的角色半身像 sprite sheet：鲍先进，同一个角色的 5 个克制写实表情，另有 1 个空白格。不是全身行走表，不是 UI 面板，不是展示海报。

== 风格声明 ==
32-bit JRPG era pixel art，PS1 / GBA-to-DS RPG 质感，清晰 chunky pixel，1px 深炭灰或深棕描边，有限但丰富的怀旧色板，轻微抖动阴影。不要平滑厚涂，不要 3D，不要照片感，不要 anti-aliasing。
时代气质是 2008 年前后中国农村小学，温暖、朴素、略晒旧。

== 用途声明：必须严格按格切片 ==
这张图会被脚本按 2 行 x 3 列均匀切割成 6 张独立贴图。你必须想象画面上有 2 行 x 3 列的隐形等分线：垂直等分线在整图宽度的 1/3 和 2/3 位置，水平等分线在整图高度的 1/2 位置。
每个肖像只能放在自己那一格的正中央，不能碰到任何隐形等分线。头发、脸、手、肩膀、外套、课本、烟雾、描边都必须完整收在本格中央 50% 的内圈区域内，所有可见像素距离本格边界至少 25% 安全距离。
如果画绿色等分线检查，绿色线必须只穿过纯洋红 #FF00FF 背景，绝对不能切到人物的脸、头发、手、肩膀、衣服、烟雾或描边。

== 画布布局 ==
- 2 行 x 3 列隐形虚拟网格，共 6 个完全相同的方形格。
- 每个格子逻辑规格 64x64，正方形 1:1。
- 整张图宽高比约 3:2。
- 背景是 100% 连续纯洋红 #FF00FF，虚拟网格不画出来。
- 肖像之间必须有很宽的纯洋红空白带。
- 不要分隔线、不要分隔条、不要 tile 边框、不要整张图外框。

== 每格摆放位置 ==
第 1 行第 1 列：严肃。肖像中心在左上格中心。
第 1 行第 2 列：凶。肖像中心在上中格中心。
第 1 行第 3 列：笑。肖像中心在右上格中心。
第 2 行第 1 列：红眼。肖像中心在左下格中心。
第 2 行第 2 列：抽烟。肖像中心在下中格中心。
第 2 行第 3 列：空白。

== 角色外观锁定 ==
参考身份：assets/art/characters/baoxianjin/raw/char_baoxianjin_concept_v2.jpg
鲍先进，45-50 岁，中国农村小学六年级班主任/数学老师。半身像只画头、肩、少量胸口，整体尺寸要小而集中，宁可留出大量洋红空白，也不要靠近格子边界。
- 方脸，宽颧骨，皮肤黝黑粗糙 #7f5a42。
- 黑短硬寸头 #1a1410，不戴眼镜，不留长胡子。
- 粗眉、额头纹、法令纹明显，嘴角自然下垂。
- 深蓝黑外套 #34373d 或墨绿外套 #3e4a3a，领口有一点粉笔灰 #d8d0b8。
- 主描边 #2a2520。
- 同一角色 5 个表情里脸型、发型、皱纹、外套领口必须一致，只改变表情和小动作。

== 6 个格子 ==
严肃：眉毛压住，眼睛直视略向下，嘴角自然下垂，像在讲台上盯人。
凶：眉头收紧，眼睛更窄，法令纹加深，但不要变成恶人或怪物。
笑：嘴角只轻微上扬，眼角皱纹放松，仍保持中年老师的压场感，不露夸张大牙。
红眼：眼白略带暗红 #9b4d4d，眼眶疲惫，嘴唇绷住，像熬夜批作业后强撑。
抽烟：一手在胸前夹细烟，灰白烟雾 #c8c0b0 只画 2-3 小段，烟雾完全收在本格中央，不能越界。
空白：这一格必须完全空白。只有纯洋红 #FF00FF 填充，没有任何图形、点、线、十字、占位符。绝对什么都不画。

== 强禁项 ==
- 不允许文字、字母、数字、姓名标签、图例、编号、水印、签名。
- 不允许 UI 框、对话框、边框、tile 描边框、画布外框。
- 不允许大汗珠、椭圆 ○○ 惊讶眼、波浪嘴、夸张漫画符号、魔法符号。
- 不允许任何元素靠近或跨过 1/3、2/3 垂直等分线和 1/2 水平等分线。
- 不允许人物变大、居中成一整张群像、跨格排列或贴近画布边缘。
- 不允许白底、灰底、纸张底；所有空白必须是纯洋红 #FF00FF。
- 不允许 anti-aliasing、模糊、柔焦、照片质感、3D 渲染。
