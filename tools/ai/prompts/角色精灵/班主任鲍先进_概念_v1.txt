单个 RPG 角色概念预览图：张老师，丁埠小学六年级班主任。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中。
- 背景为纯洋红 #FF00FF 平面色，方便后续抠图；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 Godot 顶视 RPG 角色素材的概念稿：轮廓清楚，颜色块明确，细节适合后续缩成 32x32 或 48x48 sprite。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。
- 风格与 2008 年前后中国农村小学怀旧调一致，克制、朴素、可读。

== 角色形象 ==
- 张老师，40 岁左右，中国农村小学男班主任，微胖，圆脸，有轻微双下巴。
- 黑色短发，寸头偏短，略微自然卷；不戴眼镜，不留胡子。
- 神情严肃但不凶，像负责的班主任，眉毛稍粗，眼神稳定。
- 体型约 4.4 到 4.6 头身，比学生更高更宽，肚子微微鼓起，但不要夸张肥胖。

== 服装 ==
- 深灰色中山装上衣 #4a4a48，立领，前襟有几颗小暗扣，用像素块简化。
- 黑灰长裤 #2e2e2c，黑皮鞋 #1d1b1a。
- 服装干净但有年代感，不要西装领带，不要运动服，不要现代夹克。

== 道具 ==
- 右手夹一本蓝色封面的语文课本 #2e5f9e，书本夹在身体侧前方，封面不要写真实文字，可以用 1-2 条浅色横线表示书页。
- 左手自然垂在身侧。
- 不要手机、电子屏、保温杯、教鞭。

== 禁止项 ==
- 不要眼镜，不要现代品牌 logo，不要英文字母。
- 不要把角色画成校长、官员、保安或西装商务人士。
- 不要过度喜剧化，不要夸张大肚子。
