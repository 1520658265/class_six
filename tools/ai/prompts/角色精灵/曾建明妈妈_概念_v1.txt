单个 RPG 角色概念预览图：曾建明妈妈。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 64%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年古碑镇家庭妇女/干部气质，整洁、热情、生活条件较好。

== 角色形象 ==
- 曾建明妈妈，40 岁左右，短发烫卷，气质像镇上妇女干部或教师家属。
- 表情亲切，会招呼客人吃饭，笑得自然。
- 身形中等，不要过度时髦。

== 服装 ==
- 暗红或棕色毛衣 #8a4a42，可配灰色外套或西装外套 #6b6760。
- 深色长裤，干净布鞋。
- 整体比农村家长更整洁，但仍符合 2008 年小镇审美。

== 道具 ==
- 可手里端一个碗或水果盘，表现招待客人。
- 不要现代手提包、手机、奢侈首饰。

== 配色 ==

- 头发深褐黑 #241c18
- 主描边 #2a2520
- 毛衣暗红 #8a4a42
- 外套灰褐 #6b6760

- 碗白 #e8e2d4
- 水果暖黄 #d8a540

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母。
- 不要画成城市贵妇、服务员、商务人士或过度时髦。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
