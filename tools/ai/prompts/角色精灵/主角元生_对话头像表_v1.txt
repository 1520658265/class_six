一张主角元生的像素风对话头像表，用于 RPG 对话框。不是全身立绘，不是 walking sprite sheet。

== 画布布局 ==
- 一张横向排布的头像表，包含 6 个方形头像格，单行排列。
- 每个头像格表现同一个角色元生的不同表情。
- 背景统一纯洋红 #FF00FF，方便后处理抠图；不要文字、不要标签、不要编号、不要边框、不要水印。

== 风格 ==
- 16-bit JRPG / GBA 像素头像风格，清晰 chunky pixel，1px 深描边。
- 每个头像只画头部到上胸，突出发型、眼神、红领巾和领口。
- 同一角色五官、发型、配色必须完全一致，只变化表情。

== 角色基准 ==
元生，2008 年中国农村小学六年级男生，校园少年版酷帅杀马特。
- 黑发 #1a1410，斜刘海遮住右眼，只露左眼
- 肤色 #e8b890
- 米白短袖 #f2ead8，深蓝领口 #1f2c46
- 红领巾 #c83828
- 左眼半眯，整体有点酷、有点要面子

== 6 个表情，从左到右 ==
1. 普通：平静，轻微装酷
2. 不服气：眉毛压低，嘴角绷住
3. 得意：轻微抬眉，嘴角一点点上扬
4. 紧张嘴硬：眼神飘一下，但还在装镇定
5. 惊讶：眼睛睁大一点，但仍是同一角色
6. 被看穿：有一点尴尬，但不是羞怯哭相

== 禁止项 ==
- 不要出现自卑、胆怯、苦相、委屈落泪。
- 不要现代潮牌、耳钉、染发、成人妆感。
- 不要多个不同角色混进头像表。
