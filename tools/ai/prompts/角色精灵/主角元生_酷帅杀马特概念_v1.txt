单个主角概念预览图，用来评估角色是否有 2008 年校园主角感和轻微杀马特气质。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中。必须只有一个人，不要并排三人，不要多个版本，不要角色阵列，不要 turnaround。
- 背景是纯洋红 #FF00FF 平面色，用于后处理抠图；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 16-bit JRPG / GBA 风格像素画，chunky pixel，清晰 1px 深色描边，不要平滑插画感，不要写实厚涂。
- 角色应像可缩到 32x32 sprite 的紧凑 RPG 设计稿：大轮廓清楚，头稍大，身体短，比例紧凑，细节克制，不要堆太多装饰。角色占画面高度约 65%，不要画成高挑时装插画。

== 角色核心气质 ==
小明，1996 年生，2008 年中国农村小学六年级男生，12 岁左右。重点是“清爽、有主角感、仍然是校园少年”：有一点拽、有一点自信、有一点杀马特风潮影响，但不是成人潮人，不是混混，不是高中生。

== 头发和脸 ==
- 黑色头发 #1a1410，不染发、不挑染。
- 校园少年版杀马特斜刘海：刘海从右上斜向左下盖住右眼，只露出左眼；刘海边缘分层，发梢略微外翘，整体比普通锅盖头更利落、更有辨识度。
- 后脑勺和两侧头发不要太长，不要变成女性长发。
- 脸型略窄，不要圆胖脸；肤色 #e8b890。
- 露出的左眼是半眯的黑色像素眼，眉毛略斜，表情有点酷、有点不服气；嘴角可以有非常轻微的自信小表情。
- 不要耳钉、项链、纹身、化妆。

== 身材比例 ==
- 六年级校园少年的紧凑体型，约 3.5 到 3.8 头身；头明显偏大，身体短，四肢比例紧凑。
- 比普通 NPC 清爽、有主角感，但不能像初中生或高中生；肩膀窄，轮廓仍有低龄校园感。不要高挑比例、不要模特比例。
- 站姿放松，右手插短裤口袋，身体轻微偏重心，看起来有点酷，但不要成熟冷酷。

== 服装 ==
- 上衣：米白色短袖 T 恤 #f2ead8，深蓝色领口和袖口 #1f2c46，胸前有一个很小的深蓝抽象闪电/斜杠图案；图案必须没有文字、没有品牌、没有英文字母。
- 红领巾：鲜红 #c83828，打得小一点，挂在短袖外面，不要大到抢掉脸和发型。
- 下装：浅蓝短牛仔裤 #5f8fb8，长度到膝盖上方，裤脚有卷边，深蓝描边 #2a3850。
- 鞋子：白色低帮运动鞋 #f0f0e8，黑色鞋底和深色描边 #2a2520，看起来比黑布鞋更清爽。
- 绝对不要旧棉外套、不要中山装、不要黑布鞋、不要棕灰土衣服、不要破烂脏衣服。

== 配件 ==
- 不背任何包，不要斜挎包，不要双肩包，不要书包带；让上半身线条保持清爽。
- 左手腕戴一个黄色塑料手环 #f0c828，小而亮，是 2008 年学生感点缀；不要画成智能手表。
- 右手始终插在短牛仔裤口袋里，左手自然垂下。

== 风格边界 ==
- 要有主角感，但必须保留小学六年级年龄感：头明显偏大、脸更稚气、比例紧凑。
- 严禁生成多个角色、多个副本、三连角色、角色阵列。
- 要有杀马特影响，但不要夸张成成人非主流、朋克、摇滚歌手或现代网红。
- 要清爽、有主角辨识度，不要土旧棉衣，不要过度乡土化。
- 不要斜挎包、双肩包、书包带、手机、耳机、电子屏幕、现代智能设备、品牌 logo、英文文字。
