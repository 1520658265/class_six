单个 RPG 角色概念预览图：鲍先进，丁埠小学六年级班主任。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 68%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。
- 2008 年前后中国农村小学怀旧调，朴素、压场、可信。

== 角色形象 ==
- 鲍先进，45-50 岁，中国农村小学班主任/数学老师，偏壮，肩宽，脸方，皮肤黝黑粗糙。
- 眉毛粗，法令纹和额头纹明显，神情严肃、压得住场，但不是恶人脸。
- 头发黑短，略硬，寸头偏短，不戴眼镜，不留长胡子。
- 身体比学生大一圈，站姿稳，像长期站讲台的人。

== 服装 ==
- 深色夹克或中山装风外套，颜色偏深蓝黑 #34373d 或墨绿 #3e4a3a。
- 冬天可带旧军绿色棉大衣感，但轮廓别臃肿。
- 深色长裤 #2e2e2c，黑布鞋或旧皮鞋 #1d1b1a。
- 袖口和领口可以有一点粉笔灰脏痕。

== 道具 ==
- 一手夹课本或教案本，另一手可自然下垂或拿粉笔盒。
- 画面里可以有一根细烟，但不要夸张烟雾，不要现代烟盒文字。
- 不要手机、电子屏、对讲机、现代办公用品。

== 禁止项 ==
- 不要眼镜，不要现代品牌 logo，不要英文字母。
- 不要把角色画成校长、保安、官员或商务人士。
- 不要过度喜剧化，不要过度凶神恶煞。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
