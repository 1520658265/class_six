一张 P0 核心剧情道具图标 sprite sheet，2 行 x 3 列，共 6 个独立道具图标。用于 RPG 背包、剧情提示和 UI，不是场景图。

== 风格 ==
32-bit JRPG / GBA-to-DS 像素图标，清晰 chunky pixel，1px 深色描边，2008 年中国农村小学怀旧调。每个道具单独居中，背景纯洋红 #FF00FF 用于透明抠图。

== 切片用途 ==
这张图会被按 2 行 x 3 列均分裁切成 6 个独立图标。每个道具必须完整收在自己格子的中央 50% 内圈，所有可见像素距离本格边界至少 25% 安全距离。如果画绿色等分线，绿色线只能穿过纯洋红背景，不能切到道具。

== 布局 ==
- 2 行 x 3 列，整图宽高比约 3:2。
- 背景 100% 纯洋红 #FF00FF。
- 不要分隔线、不要边框、不要编号、不要文字标签。

== 六个道具，从左到右、从上到下 ==
1. 铝饭盒：圆形铝制饭盒，盖子有白胶布名签但不要可读字，边缘磕碰凹痕。
2. 辣条：透明小包装，红油面筋条，包装用红黄色块暗示，不要品牌字。
3. 跳跳糖：拇指大小彩色小纸袋，卡通色块，不要可读字。
4. 5 毛硬币：铜色梅花币，可旁边放一两枚小铝色 1 毛，但主角是 5 毛。
5. 火力少年王悠悠球：透明塑料外壳，内部彩色转盘，红色尼龙绳。
6. 赊账本：32 开硬皮红色本子，封面红双喜色块暗示，不要文字，露出圆珠笔。

== 禁止项 ==
- 不要文字、字母、数字、品牌 logo、水印、签名。
- 不要白底、灰底、纸张底；所有空白必须纯洋红 #FF00FF。
- 不要让任何道具跨格或靠近等分线。
