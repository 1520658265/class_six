主角元生的横版战斗本体动作表，用于 2D 横版格斗 RPG。**只画角色身体动作，不画任何武器，不画多人。**

== 画布布局 ==
- 一张动作表，横向排布 8 个独立动作格，或者 2 行 × 4 列共 8 格。
- 每个格子里是同一个角色元生的一个完整侧视战斗 pose。
- 背景统一纯洋红 #FF00FF，方便后处理抠图；不要文字、不要标签、不要编号、不要边框、不要水印。

== 用途 ==
这是“横版侧视战斗本体”设计稿，不是地图 walking sheet，不是对话头像。
重点是建立元生在横版战斗里的身体语言：爱装酷、要面子、动作不专业但有劲头。

== 风格 ==
- 16-bit JRPG / GBA 像素风，chunky pixel，清晰 1px 深色描边。
- 角色为全身侧视或轻微 3/4 侧视，适合后续拆成横版战斗 sprite。
- 小学生打架，不是武侠，不是格斗冠军；动作应偏推搡、冲劲、笨拙、逞强。

== 角色基准 ==
元生，12 岁左右，2008 年中国农村小学六年级男生。
- 黑发 #1a1410，酷帅杀马特斜刘海
- 米白短袖 #f2ead8，深蓝领口袖口 #1f2c46
- 红领巾 #c83828
- 浅蓝短牛仔裤 #5f8fb8
- 白色低帮运动鞋 #f0f0e8
- 黄色塑料手环 #f0c828
- 体型仍然是小学六年级，不要画成成年格斗角色

== 动作，从左到右 ==
1. battle_idle：侧身站立，肩膀微绷，嘴硬装镇定
2. run：向前冲
3. brace：咬牙顶住，像准备挨一下
4. push：双手或单肩向前猛推
5. punch：不标准的小学生出拳
6. swing_body：通用挥击身体动作，手里不拿武器，给后续书包/扫帚/饭盒复用
7. hurt：被打到，身体后缩
8. getup：从摔倒后撑起来

== 关键规则 ==
- 所有动作都不带武器，手里不要画书包、扫帚、饭盒、悠悠球。
- 这是“本体动作层”，后续武器会单独贴到手部挂点。
- 表情不要苦相，不要哭，不要怂；可以紧张，但仍然嘴硬。
- 不要生成多个不同造型版本，必须是同一个元生。

== 禁止项 ==
- 不要现代耳机、手机、潮牌 logo。
- 不要把动作画成拳皇式专业格斗招式。
- 不要夸张特效火焰、雷电、魔法。
