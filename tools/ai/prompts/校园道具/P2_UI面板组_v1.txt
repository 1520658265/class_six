一张 P2 UI 组件 sprite sheet，2 行 x 2 列：时代切片收集面板、日记本界面、结局画面模板、回忆闪回效果。

== 风格 ==
像素风 UI，32-bit JRPG / GBA-to-DS 质感，作业本、邮票册、旧照片怀旧风。背景纯洋红 #FF00FF。

== 切片用途 ==
按 2 行 x 2 列均分裁切。每个 UI 完整居中，绿色等分线只能穿过洋红背景。

== 四个组件 ==
第 1 行第 1 列：时代切片收集面板。仿邮票集邮册，14 个格子，部分彩色邮票，部分灰色轮廓，不要文字。
第 1 行第 2 列：日记本界面。翻开的横线本，左页列表色块，右页回忆文字区但不要可读文字。
第 2 行第 1 列：结局画面模板。像素定格场景占大块，下方结局名位置用模糊手写线条表示，不要可读文字。
第 2 行第 2 列：回忆闪回效果。旧照片边框、低饱和、噪点、暗角，用一小段山路回忆画面表达。

== 禁止项 ==
不要可读文字、数字、现代 UI、霓虹、白底、灰底、水印。
