单个 RPG 角色概念预览图，参考一张现实照片中的年轻男性形象：短黑发、清爽脸型、穿红色华丽龙纹戏服/官服，手里拿着黑色圆顶官帽。只生成一个角色，不生成 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中。
- 背景为纯洋红 #FF00FF 平面色，方便后续抠图；不要真实建筑背景、不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit JRPG / GBA 风格像素画，清晰 chunky pixel，1px 深色描边。
- 可作为 Godot 顶视 RPG 角色素材的概念稿：轮廓清楚，颜色块明确，细节适合后续缩成 32x32 或 48x48 sprite。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。

== 角色形象 ==
- 年轻中国男性，清爽端正，短黑发，发型自然向上梳理，鬓角干净。
- 表情温和自信，眼睛微微向下看，嘴角有很轻的笑意。
- 身材修长但不要成人模特夸张比例，适合 RPG 主角或重要 NPC。
- 重点是“现实照片转为 RPG 角色”，保留人物气质，不追求照片级肖像复制。

== 服装 ==
- 穿一件红色华丽戏服/龙纹官服，主色为鲜红 #c92f22。
- 衣服是宽袖长袍，肩部和袖口有金色、蓝色、白色的云纹刺绣装饰。
- 胸前有一个简化的金色龙纹图案，龙纹用像素块概括，不要画得太密；必须能看出是中国传统戏服/龙袍风格。
- 领口有深蓝色圆领边 #1f2742，边缘带金色回纹装饰，但图案要简化成少量像素块。
- 金色刺绣 #d6a83a，蓝色点缀 #2f6ca3，白色高光 #f3e8c8。
- 服装整体华丽、喜庆、舞台感强，但仍然要像 RPG 像素角色。

== 道具 ==
- 一只黑色圆顶官帽/戏服帽作为道具，抱在左手或夹在左臂下。
- 官帽主色黑色 #1c1715，带一圈金色边线和一个小金色装饰。
- 不要矿泉水瓶，不要手机，不要现代背包，不要现代建筑背景。

== 禁止项 ==
- 不要照片背景里的玻璃建筑、植物、矿泉水瓶。
- 不要现代 logo、英文字母、屏幕、耳机、手表。
- 不要真实摄影风格，不要 AI 写实人像。
- 不要把角色画成皇帝坐姿；必须是站立全身 RPG 角色。
- 不要过度复杂刺绣导致缩小后糊成一团，龙纹和云纹要概括。
