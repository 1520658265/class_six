单个 RPG 角色概念预览图：大壮，丁埠小学六年级男同学。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 65%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续抠图；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 Godot 顶视 RPG 角色素材的概念稿：轮廓清楚，颜色块明确，细节适合后续缩成 32x32 或 48x48 sprite。不要画成半身头像、近景肖像或海报大立绘。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。
- 2008 年前后中国农村小学怀旧调，朴素但有辨识度。

== 角色形象 ==
- 大壮，12 岁左右，中国农村小学六年级男生，是主角同班男同学。
- 体型比主角略胖、结实，约 3.7 到 3.9 头身，肩膀圆，脸圆，站姿憨厚。
- 黑色短发偏厚，有点毛躁，发际线自然。
- 鼻头略红，脸颊健康，表情直率、有点憨、有点讲义气。
- 不要画得太帅，不要成人化，不要像小混混。

== 服装 ==
- 旧棕色毛衣 #7a4d2a，针织感用少量像素暗线表现，袖口略深。
- 深灰长裤 #3b3a38，裤脚略宽。
- 解放鞋/老式布鞋，鞋面灰绿色 #5e654d，鞋底深色 #2a2520。
- 戴红领巾 #c83828，结在胸前，大小适中。
- 不背书包，不戴帽子，不戴手表。

== 禁止项 ==
- 不要现代品牌 logo、英文字母、电子设备。
- 不要黑色皮鞋、运动潮鞋、牛仔短裤。
- 不要过度丑化或喜剧化，必须仍然可爱可信。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
