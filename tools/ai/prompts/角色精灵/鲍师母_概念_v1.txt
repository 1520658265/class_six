单个 RPG 角色概念预览图：鲍师母，丁埠小学校内小卖部老板娘。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 64%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后中国农村小学怀旧调，温和、朴素、带点烟火气。

== 角色形象 ==
- 鲍师母，40 多岁，中国农村小学小卖部老板娘，微胖，圆脸，笑起来眼睛眯起。
- 头发黑色，向后扎起，用黑皮筋束住，别得整齐。
- 神情亲切、会算账、能管事，不要像城里售货员。

== 服装 ==
- 碎花围裙或深色围裙，底色偏深蓝灰 #4a5662 或深褐灰 #5a5148。
- 里面穿深色毛衣或棉线衣，冬天可加棉马甲。
- 下身深色长裤或长裙，鞋子朴素。

== 道具 ==
- 右手拿翻开的账本或圆珠笔，左手可拿铝壶或递东西。
- 可以带一点小卖部气质的小物件，但不要摆满货架。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要过度漂亮或年轻化。
- 不要高跟鞋，不要现代围裙，不要超市收银员风。
