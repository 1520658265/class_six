主角元生的横版战斗“校园物件”分层表，用于 Godot 可替换武器层。重点是生活道具，不是武器店道具。**只画物件，不画人物，不画手，不画 UI 图标。**

== 画布布局 ==
- 严格 2 行 × 3 列，共 6 个格子；不要变成 3 行 × 3 列。
- 每个格子只放一个独立物件，第 6 格保持空白。
- 格子之间只用宽的纯洋红 #FF00FF 留白分隔，不要黑色粗边框、不要标签、不要编号、不要文字、不要水印。
- 背景必须是统一纯洋红 #FF00FF，方便后处理抠图；物件边缘不要阴影投到背景上。

== 风格 ==
- 16-bit JRPG / GBA 像素风，chunky pixel，清晰 1px 深色描边。
- 物件尺寸控制在“小学六年级男生拿得动”的比例，适合贴到角色手部挂点。
- 所有物件都是 2008 年中国农村小学常见物品，朴素、旧一点、生活化。

== 内容，从左到右、从上到下 ==
1. 旧帆布单肩书包：绿色或灰绿色，软塌，有翻盖、肩带、少量磨损补丁；像学生书包，不像登山包或电竞背包。
2. 草扫帚：木杆 + 黄草束头，普通校园值日扫帚；像扫地工具，不像长枪、法杖或兵器。
3. 扁平铝饭盒：浅银色、低矮椭圆或圆角长方形饭盒，有盖子、卡扣或小提耳；必须是学生蒸饭用饭盒，不要画成水桶、锅、保温桶、饭煲。
4. 塑料悠悠球：蓝色或透明塑料双圆盘，中间轴明显，带一段红色棉线；像 2008 年小学生玩具，不要画成刀、剑、飞镖。
5. 黑板擦：小矩形，木头底板 + 黑色或深灰毡面，粉笔灰痕迹；像教室黑板擦，不像砖头。
6. 空挂点：这一格完全空白，只保留纯洋红背景，不画十字、不画占位符、不画任何物体。

== 必须避免 ==
- 绝对不要出现长剑、短剑、刀、匕首、金属刃、武士刀、长枪、斧头、锤子、棒球棍、枪械、盾牌。
- 不要生成第二个饭盒来填空格；第 6 格必须空白。
- 不要现代潮牌 logo、英文字母、霓虹涂装、科幻机械细节。
- 不要人物手臂、手掌、角色身体或战斗姿势。
- 如果某格无法准确表达指定物件，宁可画得简单，也不要替换成刀剑兵器。
