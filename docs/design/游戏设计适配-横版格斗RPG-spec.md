# 元生的六年级 — 2D 横版格斗 RPG 游戏设计适配文档

> 本文档基于「元生六年级故事线.md」，针对 **2D 横版格斗 RPG** 类型进行机制层适配。
> 目标参考：叙事深度参考《空洞骑士》《蔚蓝》；格斗手感参考《铲子骑士》《盐与避难所》；成长系统参考《异度之刃》轻量版。

---

## 一、核心玩法定位

### 游戏类型拆解

| 维度 | 定位 |
|------|------|
| 移动视角 | 2D 横版（左右卷轴为主，部分场景可上下层） |
| 战斗风格 | 近战格斗为主（拳脚/书包砸/扫帚），可解锁技能 |
| RPG 层 | 属性成长 + NPC 好感度 + 对话分支 + 多结局 |
| 叙事层 | 章节推进，场景切换，回忆片段插叙 |
| 基调 | 写实山区童年，不超现实，不魔法 |

### 战斗的"合理性框架"

元生是小学生，战斗必须贴合现实：
- **无器械战斗**：拳脚、推搡、摔跤、书包抡
- **环境利用**：课桌椅、扫帚、黑板擦、饭盒
- **情绪系统驱动**：胆量属性决定能否发动某些攻击
  - 胆量低 → 只能防御/逃跑
  - 胆量高 → 解锁"硬刚"、"反推"等进攻技能
- **战斗不是主旋律**：大多数冲突可以用"逃跑/对话/告状"解决，选择战斗是一种属性高的奖励路线

## 二、场景地图设计（横版化）

### 场景层级

```
丁埠小学（主地图）
├── 一楼走廊（左→右横版，带压迫感BUFF）
│   ├── 一班教室（进入触发特殊状态）
│   ├── 水房（打水/冲突高发）
│   └── 一楼出口（通往操场）
├── 楼梯口（上下层切换点，曹正东常驻）
├── 二楼走廊
│   ├── 二班教室（元生主场，靠窗第三排）
│   └── 二楼窗口（可俯瞰操场/小卖部）
├── 操场（横版大场景，运动会/广播体操/集会）
├── 男生宿舍（夜晚专属，熄灯后触发特殊事件）
└── 小卖部（宿舍右侧，特殊互动区，非战斗）

丁埠镇（外围地图）
├── 小街（探索型，几个固定摊位NPC）
├── 电话亭（剧情触发点）
└── 丁埠暴动旧址/烈士纪念碑（第六章解锁）

山路（过渡地图，行走+随机事件）
├── 回家山路（周五专属）
│   ├── 板栗树（可互动采集）
│   ├── 河边小路（体力检定）
│   └── 山顶视野点（回忆触发）
└── 元生家（古碑村，周末专属）
    ├── 堂屋（家庭对话核心区）
    ├── 茶园（劳动小游戏区）
    └── 村道（村里小伙伴互动）
```

### 场景切换逻辑

- **周内**（周一到周五）：丁埠小学全图可探索
- **周五下午**：触发"走山路回家"过渡地图
- **周末**：元生家地图解锁，小学地图暂不可用
- **周一早上**：再次回到小学，NPC状态刷新

## 三、战斗系统设计

### 3.1 基础战斗框架

```
输入：
← → 移动
↑   跳跃（部分场景双跳）
□   普通攻击（拳/推）
△   特殊攻击（需胆量值）
○   闪避/后退
×   互动/确认

战斗资源：
HP（体力条）：受伤减少，睡觉/吃饭恢复
勇气槽：胆量属性的实时表现，发动高级动作消耗
```

### 3.2 技能树（按胆量/剧情解锁）

| 技能名 | 解锁条件 | 效果 |
|--------|---------|------|
| **忍着** | 默认 | 被攻击时降低伤害，但勇气槽消耗 |
| **推开他** | 胆量 ≥ 2 | 近身反推，造成硬直 |
| **书包砸** | 胆量 ≥ 3 | 抡书包攻击，范围伤害，有CD |
| **跑！** | 默认 | 冲刺逃跑，消耗体力 |
| **顶回去** | 胆量 ≥ 4 + 第六章选B | 正面硬刚，造成大伤害但自身也受伤 |
| **爆发** | 胆量 ≥ 5 + 特殊愤怒状态 | 无敌帧 + 连击，消耗全部勇气槽 |
| **那你来啊** | 口才 ≥ 4 | 言语反击，造成曹正东"士气下降"状态 |

### 3.3 情绪状态系统

元生在战斗中受情绪影响（替代魔法/光环系统）：

| 状态 | 触发条件 | 效果 |
|------|---------|------|
| **自卑** | 心结积累高 / 钱包羞耻度高 | 攻击力 -20%，勇气槽上限 -30% |
| **护短爆发** | 曹正东欺负更弱的同学时旁观 | 攻击力 +50%，持续一场战斗 |
| **被记住了** | 鲍师母/玲玲好感高触发的被关心回忆 | HP 恢复小量，勇气槽微回复 |
| **地震那天** | 第七章特殊战斗 | 全属性提升（角色情绪高涨状态） |
| **姐姐在底** | 解锁"姐姐是底"BUFF后 | 濒死时自动触发，HP 回复至 20% |

## 四、章节战斗关卡设计

### 序章（无战斗）
纯叙事 + 移动操作教学：
- 走廊行走、与NPC对话、进入教室找座位
- 教学：移动、跳跃、互动键

### 第一章 · 蒸饭盒里的秋天
**战斗关卡：无**（重点叙事节奏建立）
- 平台关卡：食堂找饭盒（在一排相同铝盒里辨认自己的）
- 小游戏：蒸饭盒识别（类似益智小游戏）

### 第二章 · 板栗、悠悠球与王炎
**战斗关卡 1：楼梯口 — 曹正东抢钱**
```
场景：从水房打水返回，楼梯口遭遇曹正东
敌人：曹正东（中等体型，推搡型攻击）
玩家选项触发不同战斗模式：
  A. 不吭声 → 非战斗结束（元生蹲下不动，曹正东自行离开）
  B. 喊老师 → 非战斗触发（喊叫后曹正东走，但路人议论音效）
  C. 推回去 → 胆量 ≥ 3 解锁实际战斗
     └── 胜利：胆量 +2，曹正东撤退
     └── 失败：HP 大幅受损，"心结" +2

设计意图：让玩家第一次感受到"选不了战斗"的无力感
```

**探索关卡：后山打板栗**
- 横版探索，可击打板栗树收集道具
- 体力 < 2 时无法攀爬高处的大树枝

### 第三章 · 冻疮与煤炉
**战斗关卡 2：值日生被嘲 — 曹正东再次挑衅**
```
场景：教室生炉子失败，曹正东嘲笑
此处无战斗（鲍先进出现打断）
设计：演示鲍先进的"护短"功能，建立玩家对班主任的信任感
```

**环境关卡：生炉子**
- 计时小游戏：按正确顺序点火（引燃纸→煤末→大煤块）
- 失败：曹正东嘲笑触发，鲍先进介入

### 第四章 · 春节回家（无战斗）
- 探索元生家地图
- 劳动小游戏：打糍粑（节奏游戏）、贴对联（文字游戏）
- 家庭对话树：选择答父亲的问话，影响家庭线

### 第五章 · 王炎的"再见"
**战斗关卡 3：宿舍走廊 — 王炎的对质**
```
场景：王炎堵住元生
此处无实际战斗（言语冲突）
互动系统：
  - 使用"口才"技能选项（要求口才 ≥ 3）反驳王炎
  - 沉默走开（友情清零）
  - 跟着笑（维持友情但自我嫌恶 +）
```

### 第六章 · 清明上山
**战斗关卡 4：下山路 — 曹正东当众发难**
```
场景：下山路，当着王炎和一群人的面
敌人：曹正东 + 2个小弟（围观型，不参战，但失败惩罚更重）
增加机制：围观同学影响"脸面系数"
  - 战斗中受伤：脸面受损，心结 +
  - 战斗胜利：胆量 +2，曹正东被骂（鲍先进路过）
  - 逃跑：曹正东追击一段，成功跑掉则心结 +1
  - 顶回去（胆量 ≥ 4）：触发完整战斗
    └── 鲍先进出现 → 强制战斗结束 → 曹正东被骂
```

### 第七章 · 茶季与地震
**非战斗关卡：采茶小游戏**
- 横版茶园，左右采摘（计时 + 精准度）
- 体力消耗，劳动量影响母亲给零花钱的金额

**情绪关卡：5·12 地震捐款**
- 无战斗，但钱包系统核心考验
- 玩家手里的钱决定捐多少，影响多条情感线

**战斗关卡 5：防震演练 — 带低年级撤离**
```
场景：演练中需要带低年级小孩从二楼有序下楼
平台关卡：控制元生带着多个低年级孩子（跟随型AI）
障碍：拥挤人群、倒下的椅子
机制：孩子不能受伤，体力 + 胆量双重检定
胜利奖励：胆量 +2，"被指望"成就解锁
```

### 第八章 · 小升初
**Boss战：厕所 — 曹正东最终对决**
```
Boss：曹正东（完整战斗版）
场景：厕所，一对一，无人围观
Boss技能：
  - 推搡（普通攻击）
  - 恐吓（使元生进入"自卑"状态，攻击力下降）
  - 要钱（特殊技，若元生身上有钱可支付结束战斗）

结局分支：
  A. 给钱 → 战斗结束，心结永久+，解锁"沉默的元生"结局标记
  B. 报告鲍先进 → 逃出厕所跑去找老师，触发过场，解锁"被保护"结局标记
  C. 反推（胆量 ≥ 6 / 累计冲突 ≥ 3）→ 完整击败曹正东，解锁"自己站起来"结局标记
  D. 坐下来说话（口才 ≥ 5）→ 对话技能破防，揭开曹正东家庭，解锁"理解曹正东"隐藏结局
```

## 五、探索与生活系统（横版化）

### 5.1 日常探索
每天可在学校地图自由探索，与NPC互动：
- 靠近鲍师母小卖部 → 对话触发（赊账/还钱/被塞糖）
- 路过一班 → 偷听台词，慕强自卑积累
- 在宿舍等夜晚 → 解锁熄灯后特殊事件

### 5.2 道具系统（横版收集）
在地图中拾取或解锁道具，每件对应一段回忆：

| 道具 | 获取方式 | 功能 |
|------|---------|------|
| **铝饭盒** | 开学初始装备 | 可作武器（书包砸的升级版），造成更高伤害 |
| **悠悠球** | 王炎借的 | 暗器型道具，可远程攻击，使用后归还王炎 |
| **玻璃糖纸** | 圆圆给的 | 收集物，攒够解锁圆圆线回忆 |
| **大白兔糖** | 鲍师母塞的 | 消耗品，HP恢复小量 |
| **千纸鹤盒** | 圆圆毕业礼 | 触发圆圆线结局 |
| **晨光按动笔** | 姐姐带回来的 | 装备槽，口才 +1 |
| **同学录** | 第八章解锁 | 最终章道具，各NPC题字解锁对应结局 |
| **毕业证** | 终章获得 | 游戏通关标志 |
| **姜茶（杯）** | 鲍师母冬天给的 | 消耗品，恢复HP + 短暂清除"自卑"状态 |

### 5.3 小卖部系统（横版界面）
站在小卖部窗口前呼出购买界面：
- 商品排列在玻璃柜里（横向滚动）
- 元生身上钱决定可购买项
- 鲍师母好感度影响特殊商品解锁（塞糖、姜茶、赊账）
- 赊账：购买后显示"记账条"，未还清时小卖部门口有红色提示

## 六、属性与成长系统（横版适配）

### 属性对战斗的直接影响

| 属性 | 数值范围 | 战斗效果 | 叙事效果 |
|------|---------|---------|---------|
| **学习** | 1-10 | 无直接战斗影响 | 解锁鲍先进好感、一班对话机会 |
| **胆量** | 1-10 | 解锁攻击技能、战斗选项 | 影响对话是否敢回应 |
| **口才** | 1-10 | 解锁言语技能（降BOSS士气） | 影响对话分支数量 |
| **体力** | 1-10 | 影响HP上限、冲刺耐力 | 解锁劳动关卡高难度区域 |

### 胆量成长里程碑

```
胆量 1-2：只能忍/逃，无法反击
胆量 3：解锁"推开他"
胆量 4：解锁"顶回去"，第六章可触发战斗
胆量 5：解锁"爆发"
胆量 6+：解锁第八章C结局（反推曹正东）
口才 5+：解锁第八章D结局（对话解决曹正东）
```

### 经验获取方式

- **战斗胜利**：胆量 + 体力 经验
- **逃跑成功**：体力 经验
- **说出口了**（对话选项里敢回嘴）：口才 经验
- **好好答题**（考试小游戏）：学习 经验
- **劳动关卡**：体力 经验
- **与鲍师母对话**：所有属性小量经验（"被看见了"）

## 七、关卡音效与视觉节奏建议

### 战斗音效参考

- 普通攻击：钝击感，无金属，类似书包布料/肉掌声
- 曹正东推搡：沉闷撞击 + 回声（走廊/厕所环境感）
- 勇气槽激活：心跳加速声（非音乐性，生理感）
- 胜利：无庆祝音乐，只有喘气声和环境音恢复

### 横版关卡视觉层次

```
远景层：大别山轮廓（随时间变化色调）
中景层：学校建筑轮廓/山路树木
近景层：可互动场景（地板、课桌、楼梯）
前景层：飘过的烟尘/落叶/雪花（季节感）
```

### 特殊视觉处理

- **"心结"触发时**：屏幕边缘轻微暗角，BGM 音量降低
- **"姐姐是底"BUFF激活时**：画面短暂暖色调，姐姐声音碎片
- **清明扫墓场景**：黑白 + 彩色切换（回忆与现实）
- **5·12地震那天**：轻微画面震动，色调偏灰

## 八、地图流转与存档点

### 存档机制

- **固定存档点**：二班教室靠窗第三排（元生的座位）
- **临时存档点**：鲍师母小卖部窗口（好感度高时解锁）
- **隐藏存档点**：
  - 丁埠暴动旧址（第六章后解锁）
  - 元生家堂屋（春节章节）
  - 山顶视野点（走山路时驻足3秒触发）

### 死亡/失败处理

- 横版格斗 HP 归零 → "被打倒了"（不是死亡）
- 回到最近存档点，维持所有属性/道具，仅重置当前战斗
- 多次在同一战斗失败 → 游戏提示：可选择"忍让路线"（A/B选项）跳过战斗

## 九、待细化的设计问题（需与开发对齐）

1. **操控精度**：是否需要连击系统？还是单键战斗为主？（建议单键，契合小学生身份）
2. **地图规模**：丁埠小学地图是否需要完全可探索？还是分章节解锁区域？
3. **战斗频率**：每章战斗关卡数量上限建议 1-2 场，避免偏离叙事重心
4. **失败惩罚**：失败后属性是否永久变化？（建议：战斗失败不扣属性，只影响剧情台词）
5. **语言**：方言化对话气泡 vs 普通话字幕？
6. **像素规格**：主角精灵尺寸（16x16 / 32x32 / 48x48）？
