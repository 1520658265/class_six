将一张海边双人自拍转化为 RPG 游戏风格场景图。不是 sprite sheet，不是单独角色立绘，而是一张保留原照片构图和背景氛围的完整 RPG 风格图片。

== 原图内容复刻 ==
- 竖幅自拍构图，两个人靠得很近，占画面下半部分和中间。
- 左侧是一位年轻中国女性，长黑发扎在后面，额前有几缕碎发，神情自然温和，穿白色蕾丝/刺绣感上衣。
- 右侧是一位年轻中国男性，圆润脸型，神情平静轻松，穿白色 T 恤；画面右侧露出一只比 V 的手。
- 两人都面向镜头，保持自拍亲密合影感，不要夸张表情。
- 背景必须保留海边环境：浅蓝天空、青绿色海水、白色浪花、沙滩，远处有淡淡的海岸线/小岛。
- 画面上方保留一大块深色遮阳伞伞面，从顶部斜切进入画面，形成阴影边缘。

== 男生发型重点修改 ==
- 男生发型要比原图和上一版更帅、更清爽，避免齐刘海锅盖头。
- 黑色短发，整体干净自然，有层次感；顶部略蓬松，发束向右侧轻微梳起，前额是轻微侧分/碎刘海，不要整齐平直刘海。
- 露出更多额头和眉毛，鬓角干净，后颈短，轮廓利落。
- 发型是现代日常清爽短发，不要油头、背头、染发、杀马特、长发、寸头或夸张偶像发型。
- 保留男生圆润脸型和轻松表情，只改善发型气质。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 场景插画风，像 RPG 游戏中的回忆照片或剧情 CG。
- 清晰 chunky pixel 风格，1px 深色描边，颜色块明确，细节用像素块概括。
- 画面可以比 32x32 sprite 更细致，但仍然必须有像素游戏质感，不要写实照片，不要平滑厚涂，不要 3D。
- 人物脸部要保留原照片的主要气质：左侧女生清秀温和，右侧男生圆脸、清爽短发、轻松平静。
- 背景海浪和沙滩用简化像素块表现，保留明亮夏日海边氛围。

== 色彩和光照 ==
- 夏日海边高亮光照，整体明亮、清爽。
- 海水为浅青绿色和蓝绿色，浪花白色，沙滩浅米黄色。
- 人物皮肤自然偏暖，白色衣服带一点阳光阴影。
- 遮阳伞为深灰黑色，顶部形成明显暗色块，但不要压黑整张图。

== 构图要求 ==
- 保持竖幅 3:4 自拍感。
- 两个人的头部靠近，脸部在画面中下部偏中心。
- 海平线和浪花在两人背后可见。
- 不要把人物改成全身站立，不要换成卡通头像拼贴，不要变成海报排版。

== 禁止项 ==
- 不要文字、不要水印、不要 UI、不要相机 App 标记。
- 不要新增其他人物、动物、建筑、船只特写。
- 不要把背景换成森林、城市、室内或纯色。
- 不要改变为古装、校服、盔甲或奇幻职业装；保持现代海边白色日常穿搭。
- 不要生成恐怖、阴暗、夸张表情或过度美颜风格。
