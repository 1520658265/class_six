单个 RPG 角色概念预览图：小芳，丁埠小学六年级女同学。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 60%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续抠图；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 Godot 顶视 RPG 角色素材的概念稿：轮廓清楚，颜色块明确，细节适合后续缩成 32x32 或 48x48 sprite。不要画成半身头像、近景肖像或海报大立绘。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。
- 2008 年前后中国农村小学怀旧调，温和、朴素、有生活感。

== 角色形象 ==
- 小芳，12 岁左右，中国农村小学六年级女同学。
- 体型约 3.7 到 3.9 头身，头稍大，身体小，保留小学生年龄感。
- 黑色双马尾，左右各一束，扎红头绳 #c83828；刘海齐眉，有几缕细碎发。
- 脸型圆中带秀气，表情安静认真、温和，有一点害羞但不冷漠、不怯懦。
- 不要画成成人少女、偶像、古风角色或幼儿。

== 服装 ==
- 红蓝小花布棉外套，底色偏旧红 #b54b4a，蓝色小花点缀 #315f8a，少量米白花点 #e6d4bb。
- 外套略厚，有棉衣感，但轮廓不要太臃肿。
- 深蓝长裤 #2f4563。
- 红布鞋 #b73732，颜色明确偏红，深色鞋底 #2a2520；不要画成棕皮鞋。
- 戴红领巾 #c83828，结在胸前，大小适中。
- 不背书包，不拿道具。

== 禁止项 ==
- 不要现代品牌 logo、英文字母、电子设备。
- 不要裙子、校服、时装外套、高跟鞋。
- 不要过度可爱化成幼儿，也不要成人化。
- 不要冷漠表情，不要棕色皮鞋。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
