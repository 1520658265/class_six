一张横版战斗武器分层表，用于主角元生的可替换武器系统。**只画武器，不画人物，不画手，不画多人。**

== 画布布局 ==
- 一张武器表，建议 2 行 × 3 列，共 6 个独立格子。
- 每格只放一个武器或一个武器状态。
- 背景统一纯洋红 #FF00FF，方便后处理抠图；不要文字、不要标签、不要编号、不要边框、不要水印。

== 风格 ==
- 16-bit JRPG / GBA 像素风，清晰 chunky pixel，1px 深色描边。
- 每个武器都要适合后续独立贴到角色手部挂点。
- 尺寸不要太大，保持“小学生能拿得动”的生活物件感觉。

== 武器内容，从左到右、从上到下 ==
1. 帆布书包：旧单肩书包，软挥击型
2. 扫帚：木杆 + 草束头，长柄型
3. 铝饭盒：圆形带盖饭盒，可挥击也可投掷
4. 悠悠球：透明塑料外壳 + 红绳，远程特殊型
5. 黑板擦：木底黑板擦，小型投掷物
6. 空挂点参考：一个极简白色十字或透明占位，不是 UI，只是为了确认最后一格为空武器状态

== 时代要求 ==
- 2008 年中国农村小学常见物件，不要未来感设计。
- 书包不能是现代电竞背包，扫帚不能像武器店长枪，饭盒不能像不锈钢保温盒，悠悠球要有 2008 年塑料玩具感。

== 禁止项 ==
- 不要人物手臂，不要角色身体。
- 不要夸张武侠刀剑、棒球棍、枪械。
- 不要品牌 logo、英文字母、现代花哨涂装。
