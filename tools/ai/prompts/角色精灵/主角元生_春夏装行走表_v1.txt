单张 RPG 角色 walking sprite sheet：主角元生春夏装。用于 Godot 中按整张 raw sheet 配置行走动画，不要求切成单帧。

== 风格 ==
32-bit JRPG / PS1-GBA-to-DS 像素风，清晰 chunky pixel，1px 深色描边，2008 年安徽大别山区农村小学。背景纯洋红 #FF00FF。

== 布局 ==
- 4 行 x 4 列 sprite sheet。
- 行：朝下 / 朝左 / 朝右 / 朝上。
- 列：idle / walk1 / walk2 / walk3。
- 每格是独立 32x32 逻辑角色帧，角色完整收在格内。
- 背景和格间空白统一纯洋红 #FF00FF，不要黑格线、编号、标签。

== 角色 ==
元生，11 岁，偏瘦偏矮，皮肤偏黑，板寸/毛寸不齐整。春夏装：短袖白色或浅灰衬衫，深色短裤或深色长裤，黑布鞋/解放鞋，单肩斜挎帆布包。整体朴素、晒黑、很少正视。

== 动作 ==
16 帧里发型、肤色、衣服、书包必须一致。步行帧只变化腿和手臂小幅摆动，身体最多上下 1 像素。

== 禁止项 ==
不要现代潮牌、手机、耳机、文字、标签、编号、水印、白底、灰底。
