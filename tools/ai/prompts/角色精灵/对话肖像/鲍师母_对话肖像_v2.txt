一张 RPG 对话系统用的角色半身像 sprite sheet：鲍师母，同一个角色的 3 个克制写实表情，另有 1 个空白格。不是全身行走表，不是 UI 面板，不是展示海报。

== 风格声明 ==
32-bit JRPG era pixel art，PS1 / GBA-to-DS RPG 质感，清晰 chunky pixel，1px 深炭灰或深棕描边，有限但丰富的怀旧色板，轻微抖动阴影。不要平滑厚涂，不要 3D，不要照片感，不要 anti-aliasing。
时代气质是 2008 年前后中国农村小学，温暖、朴素、略晒旧。

== 用途声明：必须严格按格切片 ==
这张图会被脚本按 2 行 x 2 列均匀切割成 4 张独立贴图。你必须想象画面上有 2 行 x 2 列的隐形等分线：垂直等分线在整图宽度的 1/2 位置，水平等分线在整图高度的 1/2 位置。
每个肖像只能放在自己那一格的正中央，不能碰到任何隐形等分线。头发、脸、手、肩膀、围裙领口、账本、描边都必须完整收在本格中央 50% 的内圈区域内，所有可见像素距离本格边界至少 25% 安全距离。
如果画绿色等分线检查，绿色线必须只穿过纯洋红 #FF00FF 背景，绝对不能切到人物的脸、头发、手、肩膀、围裙、账本或描边。

== 画布布局 ==
- 2 行 x 2 列隐形虚拟网格，共 4 个完全相同的方形格。
- 每个格子逻辑规格 64x64，正方形 1:1。
- 整张图宽高比约 1:1。
- 背景是 100% 连续纯洋红 #FF00FF，虚拟网格不画出来。
- 肖像之间必须有很宽的纯洋红空白带。
- 不要分隔线、不要分隔条、不要 tile 边框、不要整张图外框。

== 每格摆放位置 ==
第 1 行第 1 列：微笑。肖像中心在左上格中心。
第 1 行第 2 列：心疼。肖像中心在右上格中心。
第 2 行第 1 列：算账。肖像中心在左下格中心。
第 2 行第 2 列：空白。

== 角色外观锁定 ==
参考身份：assets/art/characters/baosimu/raw/char_baosimu_concept_v1.jpg
鲍师母，40 多岁，中国农村小学小卖部老板娘。半身像只画头、肩、少量胸口，整体尺寸要小而集中，宁可留出大量洋红空白，也不要靠近格子边界。
- 微胖圆脸，皮肤 #e0b08f，笑起来眼睛眯起。
- 黑发 #1a1410 向后扎起，用黑皮筋束住，整齐朴素。
- 深蓝灰或深褐灰围裙领口 #4a5662 / #5a5148，里面深色毛衣 #3a3a3a。
- 手粗糙，有少量粗茧色块 #b88465。
- 主描边 #2a2520。
- 同一角色 3 个表情里脸型、发型、围裙领口必须一致，只改变表情和小动作。

== 4 个格子 ==
微笑：眯眼笑，嘴角轻轻上扬，神情亲切但不夸张。
心疼：眉毛略向中间收，眼睛半闭，嘴角压低一点，一只手可轻轻抬到胸前。
算账：低头看小账本，眼睛变细，手拿圆珠笔或账本边缘，嘴唇轻抿。
空白：这一格必须完全空白。只有纯洋红 #FF00FF 填充，没有任何图形、点、线、十字、占位符。绝对什么都不画。

== 强禁项 ==

- 不允许 UI 框、对话框、边框、tile 描边框、画布外框。
- 不允许大汗珠、椭圆 ○○ 惊讶眼、波浪嘴、夸张漫画符号、魔法符号。
- 不允许任何元素靠近或跨过 1/2 垂直等分线和 1/2 水平等分线。
- 不允许人物变大、居中成一整张群像、跨格排列或贴近画布边缘。
- 不允许白底、灰底、纸张底；所有空白必须是纯洋红 #FF00FF。
- 不允许 anti-aliasing、模糊、柔焦、照片质感、3D 渲染。
