单个 RPG 角色概念预览图：曾建明，丁埠小学六年级班长。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 65%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后中国农村小学怀旧调，整洁、利索、比同龄人更干净。

== 角色形象 ==
- 曾建明，11 岁左右，中国农村小学六年级男生，班长，身高略高于同龄人，身形匀称。
- 头发是理发店剪的整齐短发，黑色，轮廓干净。
- 神情认真、利落、带一点被老师信任的自觉。

== 服装 ==
- 运动外套，颜色偏深蓝或藏青 #345d8b，带简单拼色条纹，但不要现代大 logo。
- 里面白色或浅蓝衬衫 #e8f0f6。
- 深色长裤 #3b3a38，运动鞋 #5f6f83 或灰白。
- 可背双肩背包，背包朴素，不要潮牌感。

== 道具 ==
- 可手拿作业本、红笔或课本，表现班长气质。
- 不要夸张道具，不要电子设备。

== 配色 ==
- 皮肤 #e8b890
- 头发 #1a1410
- 主描边 #2a2520
- 运动外套深蓝 #345d8b
- 外套拼色浅灰 #bfc8d4
- 衬衫浅蓝白 #e8f0f6
- 长裤深灰 #3b3a38
- 运动鞋灰白 #d8d4cb

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要潮牌图案。
- 不要画得过于贵气，不要像城市精英小孩。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
