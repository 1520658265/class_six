单个 RPG 角色概念预览图：元生的姐姐。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 66%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染，不要过度平滑。
- 2008 年前后安徽山区学生气质，朴素、干净、带一点外地读书的成熟感。

== 角色形象 ==
- 元生的姐姐，15-16 岁，中国山区初中/中专女生，扎黑色马尾辫，脸清秀但不刻意打扮。
- 神情温柔、稳重，像会照顾弟弟的人，不要偶像化。
- 身形比小学生高，约 4.4 头身，仍是少年，不要成人化。

== 服装 ==
- 蓝白运动校服外套，主色深蓝 #2f5f9a，白色拼接 #e8edf2。
- 深蓝运动裤 #283f68，白色或灰白运动鞋。
- 背或提一个红白蓝条纹蛇皮袋/编织袋，像从外地学校带被褥衣物回家。

== 道具 ==
- 可以一手提红白蓝编织袋，另一手自然下垂。
- 不要手机、耳机、时尚包、现代行李箱。

== 禁止项 ==
- 不要现代品牌 logo，不要英文字母，不要精致妆容。
- 不要画成成人女性、偶像、都市白领或古风角色。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
