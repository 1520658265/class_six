一张 RPG 对话系统用的角色半身像 sprite sheet：鲍师母，同一个角色的 3 个克制写实表情，另有 1 个空白格。不是全身行走表，不是 UI 面板，不是展示海报。

== 风格声明 ==
32-bit JRPG era pixel art，PS1 / GBA-to-DS RPG 质感，清晰 chunky pixel，1px 深炭灰或深棕描边，有限但丰富的怀旧色板，轻微抖动阴影。不要平滑厚涂，不要 3D，不要照片感，不要 anti-aliasing。
时代气质是 2008 年前后中国农村小学，温暖、朴素、略晒旧。

== 用途声明 ==
这张图会被按 2 行 x 2 列均匀切割，每个格子被切出来后单独作为一张对话肖像贴图使用。每个半身像必须完整落在自己那一格的方形区域内，头发、肩膀、围裙领口、账本、手部描边都不能越过自己的格子。角色外缘距离虚拟格子边界至少 25% 安全距离，相邻格子之间必须能看到连续的纯洋红空白。

== 画布布局 ==
- 2 行 x 2 列虚拟网格，共 4 个方形格。
- 每个格子逻辑规格 64x64，正方形 1:1。
- 整张图宽高比约 1:1。
- 背景是 100% 连续纯洋红 #FF00FF，虚拟网格不画出来。
- 不要分隔线、不要分隔条、不要 tile 边框、不要整张图外框。

== 角色外观锁定 ==
参考身份：assets/art/characters/baosimu/raw/char_baosimu_concept_v1.jpg
鲍师母，40 多岁，中国农村小学小卖部老板娘。半身像只画头、肩、少量胸口。
- 微胖圆脸，皮肤 #e0b08f，笑起来眼睛眯起。
- 黑发 #1a1410 向后扎起，用黑皮筋束住，整齐朴素。
- 深蓝灰或深褐灰围裙领口 #4a5662 / #5a5148，里面深色毛衣 #3a3a3a。
- 手粗糙，有少量粗茧色块 #b88465。
- 主描边 #2a2520。
- 同一角色 3 个表情里脸型、发型、围裙领口必须一致，只改变表情和小动作。

== 4 个格子，从左到右、从上到下 ==
第 1 行第 1 格，微笑：眯眼笑，嘴角轻轻上扬，神情亲切但不夸张。
第 1 行第 2 格，心疼：眉毛略向中间收，眼睛半闭，嘴角压低一点，一只手可轻轻抬到胸前。
第 2 行第 1 格，算账：低头看小账本，眼睛变细，手拿圆珠笔或账本边缘，嘴唇轻抿。
第 2 行第 2 格，空白：这一格必须完全空白。只有纯洋红 #FF00FF 填充，没有任何图形、点、线、十字、占位符。绝对什么都不画。

== 强禁项 ==
- 不允许文字、字母、数字、姓名标签、图例、编号、水印、签名。
- 不允许 UI 框、对话框、边框、tile 描边框、画布外框。
- 不允许大汗珠、椭圆 ○○ 惊讶眼、波浪嘴、夸张漫画符号、魔法符号。
- 不允许账本、手、肩膀、描边超出自己的格子。
- 不允许白底、灰底、纸张底；所有空白必须是纯洋红 #FF00FF。
- 不允许 anti-aliasing、模糊、柔焦、照片质感、3D 渲染。
