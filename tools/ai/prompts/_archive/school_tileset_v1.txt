A single pixel-art TILESET SHEET for a 1990s rural Chinese village elementary school RPG map.

CANVAS LAYOUT (very important):
- One large square canvas divided into an 8 columns x 8 rows grid = 64 cells.
- Every cell is the same size and perfectly aligned to the grid. No cell extends into another cell.
- Each cell contains exactly ONE asset, centered, drawn so it would tile/scale down cleanly to a 32x32 pixel sprite.
- Solid magenta #FF00FF fills:
    a) all space between cells (treat as 1-cell-equivalent gutter — but actually NO gutter, cells are flush; magenta fills any in-cell area not covered by the asset itself).
    b) all transparent / empty regions inside each cell (so chroma-key can later extract each tile).
- Do NOT draw cell numbers, grid lines, captions, frames, borders, watermarks, or any text labels in or around the cells. The grid is implicit through the layout of the assets.

GLOBAL STYLE (applies to every cell):
- 16-bit JRPG era pixel art (FF6, Chrono Trigger, EarthBound, Eastward).
- Readable chunky pixels — render so each cell visibly corresponds to a 32x32 pixel sprite scaled up (i.e. clearly blocky pixels, not smooth painting).
- Clean 1-pixel dark outlines on standalone objects (trees, benches, etc.). No outline on seamless ground tiles.
- Flat colors with subtle dithered shading. NO anti-aliasing. NO smooth gradients.
- Cohesive color palette across the whole sheet:
    grass green = muted olive/sage; foliage = deep pine green with hint of teal;
    dirt/road = warm yellow-brown ochre; sand = pale sandy tan;
    brick = warm terracotta red with darker mortar;
    roof tile = cool weathered grey with hint of blue-green;
    ceramic facade tile = ochre/sandy tan with light grey grout;
    water = dusty muted teal;
    wood = warm light tan / aged pine;
    accent = faded cobalt blue, painted-sign red.
- Consistent lighting direction across all cells (top-left light source) for assets that have shading.
- All Chinese characters in row 3 (丁, 埠, 小, 学) must be correctly written, bold, clearly legible, painted red on the ochre ceramic-tile facade background.
- No people, no animals, no UI elements, no text other than the four required Chinese characters.
- Background outside any asset within each cell is solid magenta #FF00FF (chroma-key alpha).
