一张 P0 章节标题卡与时代切片获得动画资源 sheet，1 行 x 2 列，共 2 个独立 UI/特效资源。

== 风格 ==
像素风 UI，32-bit JRPG / GBA-to-DS 质感，2008 年中国农村小学怀旧文具与邮票风。清晰 chunky pixel，1px 深色描边。

== 切片用途 ==
这张图会被按 1 行 x 2 列均分成 2 个独立资源。每个资源必须完整收在自己格子的中央区域，不能跨格。如果画绿色等分线，绿色线只能穿过纯洋红背景，不能切到资源。

== 布局 ==
- 1 行 x 2 列，整图宽高比约 2:1。
- 背景 100% 纯洋红 #FF00FF。
- 不要分隔线、不要边框、不要编号。

== 两个资源 ==
第 1 列：章节标题卡。黑色全屏卡片缩略图，中央有白色手写体大字位置，用抽象白色笔画块表示，不要可读文字；下方有小字位置，用短白线暗示；边缘微暗。
第 2 列：时代切片获得动画关键帧。一本邮票册角落，一枚彩色邮票从中央飞入右上角，红色印章盖下的视觉冲击，使用红印章形状但不要可读字。

== 禁止项 ==
- 不要生成可读汉字、英文字母、数字。
- 不要现代 UI、霓虹科幻、闪亮手游抽卡风。
- 不要白底、灰底；所有空白必须纯洋红 #FF00FF。
