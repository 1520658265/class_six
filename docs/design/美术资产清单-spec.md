# 美术资产清单（32×32 像素 · 横版格斗 RPG）

> 基于《元生六年级故事线 v2.1》整理
> 风格：像素风，32×32 基础网格，2D 横版
> 历史参照：2008–2009 年安徽省六安市金寨县古碑镇丁埠村级集镇及周边自然村

---

## 一、角色精灵（Character Sprites）

### 主角：元生

**外观基准**：11 岁山区男孩，偏瘦偏矮，皮肤偏黑（常年在外面跑山路晒的）。头发是母亲用推子在家剪的板寸/毛寸，不齐整。表情多数时候是低头或侧看，很少正视。

| 服装状态 | 视觉描述 | 使用章节 |
|---------|---------|---------|
| **秋装（默认）** | 洗得发白的蓝色或灰色长袖 T 恤（可能是姐姐穿旧的），深色涤纶长裤，黑色布鞋/解放鞋。书包是那种单肩斜挎的帆布包，磨得起毛 | 序章、1、2 |
| **冬装** | 厚棉袄（深蓝/墨绿色，袖口磨出棉花），里面套薄毛衣。手指有冻疮（红肿）。脚上是旧布棉鞋（第四章后换新棉鞋）| 3、5 |
| **春节在家** | 过年穿的"新衣服"——其实是亲戚给的旧羽绒服，相对干净整洁 | 4 |
| **春/夏装** | 短袖白色或浅灰衬衫（校服式），深色短裤/长裤 | 6、7、8、终 |

**动画帧集**：idle / walk / run / 被推倒-爬起 / 推人 / 蹲下递东西 / 背书包走路 / 写字（趴课桌）/ 采茶（弯腰掐叶）/ 爬树 / 默哀站立 / 淘米洗饭盒 / 甩悠悠球（笨拙）

### 核心 NPC（需完整动画）

**鲍先进（班主任 / 数学老师）**
- 外观：45-50 岁，中等身材偏壮，方脸，皮肤黝黑粗糙（民转公那一代乡村教师的典型面相）。常年穿深色夹克或中山装外套，冬天加一件军绿色棉大衣。手指发黄（烟熏）。讲课时粉笔灰沾满袖口
- 标志物：手里经常夹一根红梅/黄山牌香烟；讲台上放搪瓷茶缸（印着"奖"字或红双喜）
- 动画帧：idle / 讲课（拍黑板粉笔灰飞）/ 骂人（手指指人）/ 靠墙抽烟 / 点煤油灯 / 拎学生耳朵 / 背对学生站着（红眼）/ 拍肩膀

**鲍师母（校内小卖部老板娘）**
- 外观：40 多岁，微胖，圆脸，头发扎在脑后用黑色皮筋绑着。穿碎花围裙或深色毛衣，冬天套一件棉马甲。笑起来眼睛眯成缝
- 标志物：围裙口袋里插一支圆珠笔（记账用）；柜台上永远摆着一本翻烂的账本
- 动画帧：idle（站柜台后）/ 算账（低头翻本子）/ 递东西（从柜台伸手）/ 倒姜茶（大铝壶）/ 挥手送别 / 塞烤肠

**曾建明（班长）**
- 外观：11 岁，身高中等偏上（营养比元生好），皮肤比元生白一些（不用干农活）。头发是理发店剪的，整齐的短发。五官端正但不突出，气质是"干净利索"——衣服虽然也不贵但总是整洁的
- 服装：秋冬穿运动外套（安踏/特步那种，镇上买的），里面是白色或浅蓝衬衫。脚上是运动鞋（不是布鞋）。书包是双肩背包
- 动画帧：idle / walk / run / 招手 / 递东西 / 写字 / 翻身睡不着（宿舍夜）/ 大喊（挡人）/ 坐电脑前 / 拍人后背

**王炎（同班同宿舍）**
- 外观：11 岁，瘦长脸，眼睛灵活。前期和元生差不多穿着；后期（第五章起）开始穿从苏州带回的"城里货"——印花 T 恤、牛仔裤、白色旅游鞋
- 标志物：前期是火力少年王悠悠球；后期是山寨 MP3 手机（翻盖/直板，彩屏）
- 动画帧：idle / walk / 甩悠悠球 / 砸水杯 / 堵人（叉腰）/ 低头道歉 / 骂人

**曹正东（贯穿 BOSS）**
- 外观：11 岁但看着像十三四，偏壮偏高，脸上有一道旧疤（摔的）。皮肤黑，指甲缝里有泥。衣服总是脏的——不是没衣服穿，是没人管他换洗。冬天穿一件明显大了两号的旧棉袄（可能是他爸留下的）
- 标志物：无固定道具，但经常手插裤兜、靠墙站
- 动画帧：idle（靠墙/叉腰）/ walk / 拦路（伸手）/ 推人 / 抢东西 / 推自行车（温柔）/ 弯腰擦裤腿 / 转身走 / 战斗姿态

**姐姐**
- 外观：15-16 岁，扎马尾辫，面容清秀但不刻意打扮。穿学校发的蓝白运动校服（外地中学的），背一个大编织袋（装被褥衣物回家用的那种红白蓝条纹蛇皮袋）
- 动画帧：idle / 递东西 / 剥豆子 / 挥手上车 / 问话（侧身看元生）

**父亲**
- 外观：40 岁左右，瘦而结实，常年劳作的体型。面容沉默寡言写在脸上——眉头常年微皱。穿深色旧夹克或灰色中山装，裤腿常年卷着（干农活习惯）。手粗糙，指节粗大
- 标志物：嘴里叼烟（自己卷的旱烟或最便宜的红梅）
- 动画帧：idle / 打糍粑（举石锤）/ 抽烟看电视 / 拍肩膀 / 走山路（背篓）/ 叹气

**母亲**
- 外观：38-40 岁，中等身材，头发用黑色发卡别在耳后。穿碎花棉布上衣或深色毛衣，系围裙。手上常年有茧和裂口（冬天尤其明显）
- 动画帧：idle / 炸馓子（灶台前）/ 塞零钱 / 背过身擦眼睛 / 端碗 / 收拾行李（蛇皮袋）

### 次要 NPC

**胡晓东**：一班成绩最好的男生，11 岁，戴眼镜（山区小学戴眼镜的很少，说明看书多）。穿着比元生好但不夸张——运动服+运动鞋，气质是"镇上孩子"。动画：idle / 笑 / 点头 / 递书

**张磊**：一班男生，留守儿童但成绩好。外观跟元生差不多朴素，沉默内向。动画：idle / 点头 / 远望（看车）

**李静**：一班女生，扎两个小辫子，穿粉色或红色外套。动画：idle / 趴桌哭 / 递东西（还纸巾+奶糖）

**门卫大爷**：60 多岁退休村民，穿旧军大衣（冬）或白色老头衫（夏），戴老花镜，坐门卫室小板凳听收音机。动画：idle / 拦人（伸手）/ 跟人聊天

**曹正东奶奶**：70 多岁，极瘦佝偻，穿深色对襟棉袄，头包黑色方巾，面色蜡黄。动画：坐自行车后座 / 咳嗽

**曾建明爸爸**：40 岁左右教师气质，戴眼镜，白衬衫。动画：坐电脑前打字 / 回头看

**曾建明妈妈**：短发烫卷（镇上妇女干部典型），穿毛衣或西装外套。动画：端碗 / 笑着说话

**外校大孩子 ×2**：13-14 岁，比元生高一头，一个皮夹克一个运动服，站姿吊儿郎当。动画：idle / 围堵站位

**低年级小孩 ×3**：7-8 岁，个矮，穿得花花绿绿，跑起来手脚不协调。动画：跟随跑

### 群众/背景人物模板

2008 年大别山区小学生穿着特征：
- **男生**：无统一校服（丁埠村小不发），穿自家旧衣，颜色偏深（灰/蓝/黑/军绿），多为兄姐穿剩。鞋以布鞋、解放鞋为主
- **女生**：比男生稍鲜艳（红/粉/花），扎辫子或短发，塑料发卡
- **村民**：男穿中山装/夹克/军大衣；女穿碎花棉布衣+围裙。老人穿对襟棉袄、包头巾
- **打工归来家长**：穿得比本地人"新"——皮夹克、牛仔裤、白运动鞋，但款式落后城市两三年

模板：男生 ×3 / 女生 ×2 / 村民 ×4 / 打工家长 ×2，各配季节换装

## 二、场景背景（Tilemap + 背景层）

### 校园场景

**二班教室（二楼）—— 元生主活动区**

- 水泥地面，部分开裂。墙面白石灰已发黄，下半截刷深绿油漆（防蹭脏）
- 木质课桌椅，两人一桌，桌面刻满字画。椅子是长条板凳
- 黑板：水泥墙上刷墨绿色漆，粉笔槽是木条钉的。上方红纸标语"好好学习天天向上"
- 讲台：水泥砌台，放粉笔纸盒+搪瓷茶缸
- 窗户：铁框玻璃窗，部分碎了用报纸/塑料薄膜糊着
- 冬季加装：教室中间铸铁煤炉，烟囱管从窗户伸出，地上煤渣
- 角落：21 寸老式彩电（长虹/海信），铁架上，平时布盖
- 后墙：学习园地（红纸手写）、值日表

**一班教室（一楼）**

- 结构同二班，但一楼采光好（窗更大更亮）
- 标语"读书改变命运"（红底黄字）
- 整体比二班整洁（暗示班风差异）
- 元生初次进入需"压迫感"视觉处理（光线偏亮、空间感偏大）

**走廊 + 楼梯口**

- 水泥栏杆+铁管扶手，漆剥落露锈
- 墙上：A4 纸打印的月考排名表，透明胶带粘
- 楼梯水泥台阶，边角磨圆，有缺角
- 旁边水房入口：一排铜水龙头（有的漏水），水泥水槽
- 空间窄，两面墙——适合"堵人"冲突场景

**男生宿舍**

- 大通铺房，4-6 张上下铺铁架床，漆面斑驳
- 每铺挂发黄白纱蚊帐，被褥花色各异（从家带的棉被）
- 床底：脸盆、竹壳热水瓶、蛇皮袋装衣物
- 地面水泥潮湿，角落霉斑
- 光源：天花板一根日光灯管（嗡嗡响）；熄灯后全黑
- 窗户铁框玻璃，冬天缝里灌风

**校内小卖部（宿舍右侧）**

- 宿舍楼一楼改的一间房，开窗口当柜台
- 水泥台面柜台，摆一排大玻璃罐（散称零食）
- 墙上木架子：方便面、文具、洗衣粉等杂货
- 小型滚轴烤肠机（4-6 根容量）
- 角落冰柜（夏天卖冰棍/雪糕，5 毛-1 块）
- 光线偏暗（柜台窗口透光 + 一盏白炽灯泡）
- 鲍师母的账本永远摊在柜台角上

**食堂（蒸饭间）**

- 非现代食堂，无打饭窗口。大房间放几个巨型铝制蒸笼（直径一米多）
- 学生自带米+饭盒，早上淘米放入，中午来取
- 地面湿滑（蒸汽凝结），墙面瓷砖贴半截（上面白石灰）
- 一排水龙头+水泥水槽供淘米
- 铝制饭盒堆叠，贴各色胶带写名字

**操场 / 旗台**

- 夯实黄土地面（下雨变泥），部分区域铺水泥
- 铁旗杆，水泥底座，国旗略褪色
- 篮球架：铁管焊制，木板篮板刷白漆（剥落）
- 围墙：砖砌约两米高，顶嵌碎玻璃（防翻墙）
- 广播喇叭：教学楼外墙铁皮喇叭，锈迹斑斑
- 角落单双杠（铁管，漆面磨光）

**校门口**

- 绿漆铁栅栏大门两扇，铁链+锁头
- 门卫室：小砖房，窗台放收音机和搪瓷杯
- 校门上方水泥牌匾"丁埠小学"（红字）
- 门外通往镇街的水泥路/土路

**厕所（第八章 BOSS 战场地）**

- 旱厕，教学楼后独立砖房一排
- 蹲坑式，矮墙隔断（不到肩高），无门
- 水泥地常年潮湿发黑
- 视觉暗示：苍蝇粒子、墙角青苔
- 空间逼仄——适合"被堵"压迫感

### 镇上场景

**丁埠小街**

丁埠不是镇政府所在地（那是古碑），只是一个村级集镇。"街"就是一条两三百米的水泥路两边排着十几间门面房：
- 两三家杂货店（卖日用品、农资、种子化肥）：卷帘门，门口堆着化肥袋和扫帚
- 一个修鞋摊（路边支的铁架子+遮阳布）
- 一家小饭馆（门口挂塑料门帘，里面三四张方桌）
- 一个公用电话亭（IC 卡电话，蓝色铁皮亭子）或者杂货店门口挂的"公用电话"牌子
- 2009 年新开的奶茶冰小摊（不是店面，是路边推车或者杂货店兼营，塑料杯+吸管）
- 路面：水泥路为主，边上有排水沟（明沟），路边有几棵梧桐/杨树
- 没有红绿灯、没有路灯（或只有一两盏昏暗的）
- 偶尔经过的车辆：摩托车、农用三轮车、偶尔一辆面包车

**镇卫生院门口**

- 两层白色瓷砖贴面小楼（90 年代建的那种乡镇卫生院）
- 门口挂红十字标志和"丁埠卫生室"或"古碑镇丁埠卫生所"牌子
- 门前水泥空地，停着几辆自行车和一辆摩托
- 一楼有挂水（输液）的窗口，能看到里面的输液架
- 整体感觉：干净但简陋

**网吧（古碑镇上，非丁埠）**

- 实际上 2008 年丁埠本身可能没有网吧，网吧在古碑镇政府所在地（骑车/摩托 15 分钟）
- 外观：临街门面房，卷帘门半开，里面昏暗。门口挂"网吧"或"网络会所"招牌（喷绘布）
- 能看到里面一排 CRT 显示器的蓝光
- 门口常站着几个抽烟的半大孩子
- 只需外观，不需内部场景

**曾建明家（古碑镇中心小学家属院）**

- 家属院：三四层的老式单元楼（90 年代建），外墙贴白色小方砖，有些脱落
- 楼道：水泥楼梯，铁扶手，墙上贴着"注意防火"之类的通知
- 客厅：约 15 平米，水磨石地面。家具是 90 年代的组合柜+布艺沙发
  - 一台联想品牌台式电脑（CRT 显示器，米白色机箱），放在靠墙的电脑桌上
  - 书架一整面墙（教辅+《读者》《故事会》+几本教育学著作）
  - 茶几上：伊利酸奶、水果盘、遥控器
  - 有 DVD 播放机+电视（比学校那台新）
- 曾建明卧室：单人床、书桌、台灯、墙上贴奖状
- 整体感觉：在丁埠/古碑算"条件好的"，但放到城市就是普通

**镇车站（古碑汽车站或路边招手停）**

- 2008 年丁埠没有正式车站。实际是路边一个候车点：一块水泥地+一个铁皮棚子（遮雨用）
- 或者就是在镇街路边等班车（中巴/面包车），招手即停
- 班车：白色或蓝色中巴车，车身脏，车顶有行李架
- 旁边可能有一两个卖茶叶蛋/包子的小摊

**巷口（战斗场景）**

- 丁埠街上两排房子之间的窄巷，宽约 1.5 米
- 两边是砖墙（红砖或灰砖），墙上有青苔和小广告（"办证""疏通下水道"）
- 地面是碎石或泥土
- 巷口有一盏路灯（白炽灯泡，昏黄）或者没有灯
- 适合"被堵"的战斗场景——两头窄，跑不掉

### 山路 / 村庄场景

**回家山路（元生每周五走的路）**

- 从丁埠到元生家（五堂组）的山间土路，单程约 3-5 公里
- 路面：黄土+碎石，雨后泥泞。有些路段是山坡上凿出来的窄路（一侧是山壁一侧是陡坡）
- 两旁植被：板栗树（秋天挂果）、松树、茶园梯田、竹林、杂草灌木
- 偶尔经过一座小石桥（跨溪流）
- 远景：连绵的大别山脉轮廓，层叠的山丘，天际线很低
- 季节变化明显：秋（黄叶+板栗）→ 冬（枯枝+薄雪+灰天）→ 春（新绿+映山红）→ 夏（浓绿+蝉鸣）
- 路上偶遇：背背篓的老人、放学的低年级小孩、偶尔一辆摩托车

**元生家（五堂组农村民居）**

- 典型大别山区农村住宅：两层砖瓦房（一楼住人二楼存粮/杂物），或者是老式土坯房
- 堂屋（正厅）：水泥地面，正中八仙桌+长条凳。墙上贴年画/领袖像/日历（那种带风景照的挂历）
  - 角落有一台 14 寸彩电（可能是二手的），放在一张方凳上
  - 冬天：堂屋中间放铜/铁炭盆（烧木炭取暖）
  - 过年：门两边贴红对联，堂屋挂红灯笼
- 灶房（厨房）：土灶台（烧柴），大铁锅，墙上挂着锅铲、蒜辫、干辣椒
- 院子：水泥晒场（晒茶叶/稻谷用），旁边有鸡笼、柴垛
- 周围：茶园梯田、几亩菜地、远处的板栗林
- 没有自来水（用井水或山泉水引的水管）；有电但经常停电

**村口**

- "村口"就是进村的那条土路拐弯处，有一棵大树（香樟或枫杨）
- 树下有一块平石头（村民聚坐聊天用）
- 路边有一辆黑色桑塔纳（打工回来的人的车，在村里很扎眼）
- 旁边可能有一个简易的小卖部（村民自家门口摆的，卖盐/酱油/烟/啤酒）
- 公用电话：不一定有亭子，可能就是村部墙上装的一台投币/IC 卡电话

**外公家**

- 跟元生家类似的农村民居，但可能更旧一些（老式土坯房）
- 院子里晒着腊肉、干菜
- 过年气氛：门口贴对联、院子里放过鞭炮的红纸屑

### 特殊场景

**丁埠暴动旧址 / 烈士纪念碑**

- 在丁埠后山坡上，沿石阶而上
- 纪念碑：水泥+大理石贴面，约 5-6 米高，正面刻"革命烈士永垂不朽"
- 碑前有一块水泥平台（供献花圈/站队用）
- 周围种松柏（四季常青），山坡上有几棵老树
- 清明时：碑前摆花圈、小白花，石阶两侧学生列队
- 整体氛围：庄重但朴素，不是大城市那种气派纪念馆

## 九、Prompt 分类与命名规范

生成用 prompt 文件按**资产类型**分类存放，文件名使用中文，方便和本清单逐项对应。最终导入 Godot 的图片、音频、数据资产仍可使用英文 ID，避免引擎导入和跨平台路径问题。

### 目录结构

```text
tools/ai/prompts/
  角色精灵/
  场景建筑/
  地表自然/
  校园道具/
  参考临时/
```

### 命名格式

```text
<资产名>_<用途或规格>_v<版本>.txt
```

示例：

```text
角色精灵/主角元生_行走表_v1.txt
角色精灵/主角元生_酷帅杀马特概念_v1.txt
角色精灵/班主任鲍先进_概念_v1.txt
场景建筑/教学楼_完整10x3_v1.txt
地表自然/地表道路_tileset_v2.txt
校园道具/校园杂物_tileset_v1.txt
```

### 使用规则

- 新写 prompt 时优先放入中文分类目录。
- 旧英文 prompt 可以保留作为历史版本，不必强制删除。
- prompt 文件名服务于人工识别，可以中文；生成出的正式资源文件名服务于工程导入，建议继续英文或资产 ID。
- 同一个资产如果有概念图、sprite sheet、tileset 三种用途，拆成多个 prompt，不混在一个文件里。
