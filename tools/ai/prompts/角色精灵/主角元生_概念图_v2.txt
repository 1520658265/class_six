单个主角概念预览图，用来确定元生的最终视觉基准。只画一个角色，不画 4x4 sprite sheet，不画多人。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中。
- 背景是纯洋红 #FF00FF 平面色，用于后处理抠图；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 16-bit JRPG / GBA 风格像素画，chunky pixel，清晰 1px 深色描边，不要平滑插画感，不要写实厚涂。
- 角色应像可缩到 32x32 sprite 的紧凑 RPG 设计稿：轮廓清楚，头稍大，身体短，比例紧凑，细节克制。

== 角色身份与气质 ==
元生，1996 年生，2008 年中国农村小学六年级男生，12 岁左右。
核心气质不是自卑敏感，而是：要面子、爱耍酷、嘴上有点拽、表面装得无所谓，但本质仍然是小学生。
要有明显主角感，但不能画成混混、初中生、高中生或成人潮人。

== 头发和脸 ==
- 黑色头发 #1a1410，不染发、不挑染。
- 校园少年版杀马特斜刘海：刘海从右上斜向左下盖住右眼，只露出左眼；刘海边缘分层，发梢略微外翘，整体利落。
- 后脑勺和两侧头发不要太长，不要女性化。
- 脸型略窄但仍有小学生稚气，肤色 #e8b890。
- 露出的左眼半眯，眉毛略斜，表情是“有点酷、有点不服气、有点装帅”，但不要阴沉，不要苦相，不要低头回避。
- 嘴角可以有非常轻微的自信感，像一个爱逞强的小学生。

== 身材比例 ==
- 六年级校园少年的紧凑体型，约 3.5 到 3.8 头身；头明显偏大，身体短，四肢比例紧凑。
- 比普通 NPC 更清爽、更有主角感，但仍然一眼能看出是小学男孩。
- 站姿放松，右手插短裤口袋，身体轻微偏重心，看起来像故意装酷。

== 服装 ==
- 上衣：米白色短袖 T 恤 #f2ead8，深蓝色领口和袖口 #1f2c46，胸前有一个很小的深蓝抽象闪电/斜杠图案；不能有文字、logo 或英文字母。
- 红领巾：鲜红 #c83828，结打得小一点，挂在短袖外面，不要抢掉脸和发型。
- 下装：浅蓝短牛仔裤 #5f8fb8，长度到膝盖上方，裤脚有卷边，深蓝描边 #2a3850。
- 鞋子：白色低帮运动鞋 #f0f0e8，黑色鞋底和深色描边 #2a2520，看起来清爽。
- 不要旧棉外套、不要黑布鞋、不要土旧宽大衣服。

== 配件 ==
- 不背任何包，不要斜挎包，不要双肩包，不要书包带。
- 左手腕戴一个黄色塑料手环 #f0c828，小而亮，是 2008 年学生感点缀；不要画成智能手表。
- 右手始终插在短牛仔裤口袋里，左手自然垂下。
