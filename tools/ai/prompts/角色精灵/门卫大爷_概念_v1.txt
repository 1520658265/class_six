单个 RPG 角色概念预览图：丁埠小学门卫大爷。只画一个角色，不画 4x4 sprite sheet。

== 画面要求 ==
- 只画一名完整全身角色，站立正面略 3/4 角度，居中；必须从头到脚完整可见，角色占画面高度约 62%，四周留出大量洋红背景。
- 背景为纯洋红 #FF00FF 平面色，方便后续分类归档；不要地面、不要阴影、不要渐变、不要纹理。
- 不要文字、不要标签、不要 UI、不要边框、不要水印、不要签名。
- 不要生成多个角色、多个版本、三视图或角色阵列。

== RPG 风格 ==
- 16-bit / early-GBA JRPG 角色概念图，清晰 chunky pixel，1px 深炭灰描边。
- 可作为 32x32 或 48x48 sprite 的概念稿：轮廓清楚，颜色块明确。
- 半正面顶视 RPG 角色比例，不要写实照片感，不要厚涂插画，不要 3D 渲染。
- 2008 年前后农村小学门卫气质，朴素、慢悠悠、熟人社会。

== 角色形象 ==
- 门卫大爷，60 多岁退休村民，瘦小，略佝偻，脸上皱纹明显。
- 戴老花镜，灰白短发，神情平和但有点爱管闲事。
- 身体比例比成年人略矮，站姿可以略弯。

== 服装 ==
- 旧军大衣 #4b5740，衣摆厚重，袖口磨损。
- 里面可露出白色老头衫或深色毛衣。
- 深色裤子，旧布鞋。

== 道具 ==
- 一手拿小收音机或搪瓷杯，另一手可自然下垂。
- 不要现代保安制服，不要警棍，不要对讲机。

== 禁止项 ==
- 不要现代保安制服，不要帽徽，不要警察感。
- 不要画成军官、校长或流浪汉。
- 不要裁切头顶、脚、手臂；不要让角色顶满画面。
